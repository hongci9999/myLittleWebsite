# 분석기법 적용 (Application of Analysis Techniques)

## 📌 핵심 요약
**회귀분석**은 독립변수(X)가 종속변수(Y)에 미치는 영향을 파악하는 분석방법입니다. 최소제곱법으로 회귀계수를 추정하고, R²로 모형 설명력을 평가합니다. **다중공선성**(독립변수 간 강한 상관)은 반드시 진단·해결해야 하는 핵심 이슈입니다. **의사결정나무**는 화이트 박스 모델로 해석이 쉽지만 과적합 방지(정지규칙·가지치기)가 중요합니다. **SVM**의 **커널함수**는 저차원→고차원 변환으로 비선형 분리를 가능하게 하는 핵심 요소입니다.

---

## 📝 상세 정리

### 1. 회귀분석 (Regression Analysis)

#### ● 기본식
$$Y = wX + \beta$$
* **X**: 독립변수 (원인)
* **Y**: 종속변수 (결과)
* **잔차(오차)**: 관측값과 예측값의 차이

#### ● 회귀계수 추정방법

| 방법 | 설명 |
| :--- | :--- |
| **최소제곱법 (=최소자승법)** | **잔차의 제곱합(SSE)**이 최소가 되는 회귀계수와 절편을 구하는 방법 |

#### ● 회귀모형 평가

| 지표 | 설명 |
| :--- | :--- |
| **R² (R-square)** | 총 변동 중 회귀모형에 의해 **설명되는 변동의 비율** (0 ~ 1). 1에 가까울수록 설명력 높음. |

---

### 2. SST, SSE, SSR

| 구분 | 의미 |
| :--- | :--- |
| **SST (Sum of Squares Total)** | **전체의 변동** — 관측값과 평균의 차이 |
| **SSE (Sum of Squares Error)** | **모형에 의해 설명되지 않는 변동** — 관측값과 회귀선(예측값)의 차이 |
| **SSR (Sum of Squares Regression)** | **모형에 의해 설명되는 변동** — 회귀선과 평균의 차이 |

#### ● 관계식
$$R^2 = \frac{SSR}{SST} = 1 - \frac{SSE}{SST}$$

> [!TIP] 시각적 이해
> **관측값 → 평균**: SST (전체 변동)  
> **관측값 → 회귀선**: SSE (설명 안 됨)  
> **회귀선 → 평균**: SSR (설명됨)  
> SST = SSR + SSE

---

### 3. 회귀분석 종류

| 종류 | 설명 |
| :--- | :--- |
| **(1) 단순회귀** | 독립변수 1개, 종속변수 1개. 선형 관계. |
| **(2) 다중회귀** | 독립변수 2개 이상, 종속변수 1개. 선형 관계. |
| **(3) 다항회귀** | 독립·종속변수가 **2차 이상** 다항식 관계. |
| **(4) 규제항 포함 회귀** | 과대적합 방지를 위한 **패널티 항** 추가. |
| **(5) 교호항 포함 회귀** | 독립변수 간 **상호작용** 포함 (예: $X_1 \times X_2$). |

#### ● 교호작용(Interaction)이란?

**교호** = 두 이상의 독립변수가 **함께** 종속변수에 영향을 줄 때, 한 변수의 효과가 **다른 변수 수준에 따라 달라지는** 현상.

* **수식:** $Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \beta_3 (X_1 \times X_2)$ — $X_1 \times X_2$가 교호항.
* **예시:** 광고비(X₁) 효과가 지역(X₂)마다 다름, 학습시간 효과가 과목 난이도에 따라 다름.

#### ● 규제항 포함 회귀 세부

| 유형 | 규제 | 패널티 |
| :--- | :--- | :--- |
| **릿지회귀 (L2)** | L2 규제 | $\sum W^2$ |
| **라쏘회귀 (L1)** | L1 규제 | $\sum |W|$ |
| **엘라스틱넷** | L1 + L2 | 둘 다 적용 |

### 4. 선형회귀분석의 가정

| 가정 | 설명 |
| :--- | :--- |
| **(1) 선형성** | 종속변수와 독립변수 간 **선형 관계** |
| **(2) 등분산성** | 오차의 분산이 **일정** (독립변수 수준에 따라 변하지 않음) |
| **(3) 정규성** | 오차가 **정규분포**를 따름 |
| **(4) 독립성** | 오차가 **서로 독립** (상관 없음) |

> [!TIP] 암기법
> **선분정독**: 선형성, 등분산성, 정규성, 독립성

### 5. 회귀 모형의 구축

#### ● 구축절차

1. 독립변수·종속변수 설정
2. 회귀계수 추정
3. **회귀계수 유의성 검정**
4. **전체 모형 유의성 검정**

#### ● 회귀 모형의 검정

| 검정 대상 | 검정 방법 |
| :--- | :--- |
| **회귀계수들이 유의미한가 (개별)** | **t검정** |
| **모형 전체가 통계적으로 유의미한가** | **F검정** |

### 6. 다중공선성 문제 ⚠️

> [!IMPORTANT] 핵심
> **다중공선성** = 독립변수들 간 **강한 상관관계**가 있는 문제. 회귀계수 추정이 불안정해지고 해석이 어려워지므로 **반드시 진단·해결**해야 함.

#### ● 진단 방법

| 방법 | 기준 |
| :--- | :--- |
| **상관계수** | 변수 간 상관계수 절댓값 **0.8 이상** → 의심 |
| **VIF (분산팽창인수)** | **10 이상** → 다중공선성 존재 판단. $VIF = 1/(1-R^2)$ |
| **조건수 (Condition Number)** | **30 이상** → 의심 |

#### ● 해결방안

독립변수 제거, **차원 축소**(PCA 등), **변수 선택**, **규제항 포함 회귀**(릿지·라쏘·엘라스틱넷)

---

### 7. 최적의 회귀 방정식 탐색 방법

| 방법 | 설명 |
| :--- | :--- |
| **(1) 전진선택법** | 변수를 **하나씩 추가**하면서 최적 방정식 탐색 |
| **(2) 후진제거법** | 변수를 **하나씩 제거**하면서 최적 방정식 탐색 |
| **(3) 단계별 선택법** | 전진선택 + 후진제거. 변수 추가 시 **벌점** 고려 |

#### ● 선택 기준 (AIC, BIC, Cp)

| 기준 | 설명 |
| :--- | :--- |
| **AIC (아카이케 정보 기준)** | 편향·분산 최적화 지점 탐색. 자료가 많을수록 부정확해질 수 있음. |
| **BIC (베이즈 정보 기준)** | AIC 보완. AIC보다 **큰 패널티** 부여. |
| **Mallow's Cp** | **SSE(잔차 제곱합)** 활용. |

> [!TIP] AIC·BIC
> **둘 다 작을수록 좋음** — 모형 적합도와 복잡도의 균형을 나타냄.

### 8. 로지스틱 회귀분석 (Logistic Regression)

**목적:** 종속변수가 **범주형**일 때, 두 집단(성공/실패 등)을 분류하는 모형.

#### ● 원리

| 단계 | 설명 | 수식 |
| :--- | :--- | :--- |
| **(1) 오즈 (Odds)** | 성공확률 / 실패확률 | $Odds = \frac{P}{1-P}$ |
| **(2) 로짓(logit) 변환** | 오즈에 **자연로그**를 취해 선형 관계로 변환 | $\ln\frac{P}{1-P} = wX + \beta$ (w>0: 확률 증가) |
| **(3) 시그모이드 함수** | 로짓의 **역함수**. 확률(0~1) 도출 | $P = \frac{1}{1+e^{-(wX+\beta)}}$ |

#### ● 검정

| 구분 | 방법 |
| :--- | :--- |
| **회귀계수 추정** | **최대우도법 (MLE)** |
| **모형 전체 검정** | **우도비검정 (카이제곱검정)** |

#### ● 최대우도법 (Maximum Likelihood Estimation, MLE)이란?

**정의:** 관측된 데이터가 나올 **확률(우도)**을 최대화하는 모수를 찾는 추정 방법.

* **우도(Likelihood):** 주어진 모수(β, w 등)에서 현재 데이터가 관측될 확률. $L(\theta | data)$
* **목표:** $L$을 최대화하는 $\theta$를 찾음 → 보통 **로그우도** $\ln L$를 최대화 (계산 편의)
* **로지스틱 회귀에서 사용 이유:** 종속변수가 0/1이라 **최소제곱법(OLS)** 적용 불가. 확률을 모델링하므로 우도 기반 추정이 적합함.

### 9. 의사결정나무 (Decision Tree) ⚠️

> [!IMPORTANT] 핵심
> **의사결정나무** = 노드 내 **동질성↑**, 노드 간 **이질성↑** 방향으로 분리하는 트리 구조 모델. 해석이 쉬운 **화이트 박스**이지만 **과적합 위험**이 높음.

#### ● 특징

* **화이트 박스 모델** — 의사결정 과정이 시각적으로 명확함
* **높은 과적합 위험** — 깊은 나무는 훈련 데이터에 과적합하기 쉬움

#### ● 과적합 방지

| 방법 | 설명 |
| :--- | :--- |
| **정지규칙 (Stopping Rule)** | 더 이상 분리하지 않고 나무 성장을 멈춤 |
| **가지치기 (Pruning)** | 일부 가지를 제거하여 과적합 방지 |

#### ● 노드 분할 방법

**분류(범주형):**

| 알고리즘 | 분할 기준 |
| :--- | :--- |
| **CHAID** | 카이제곱 통계량 |
| **CART** | **지니지수** $1 - \sum P^2$ |
| **C4.5 / C5.0** | **엔트로피지수** $-\sum P \log_2 P$ |

**회귀(연속형):**

| 알고리즘 | 분할 기준 |
| :--- | :--- |
| **CHAID** | ANOVA, F-통계량 |
| **CART** | **분산감소량** |

#### ● 지니지수·엔트로피 계산 예시

동전 5개 중 앞면 3개, 뒷면 2개일 때 ($P_{앞}=3/5$, $P_{뒤}=2/5$):

* **지니지수:** $1 - (3/5)^2 - (2/5)^2 = 12/25$
* **엔트로피:** $-\frac{3}{5}\log_2\frac{3}{5} - \frac{2}{5}\log_2\frac{2}{5}$

> [!TIP] 지니·엔트로피
> **0에 가까울수록 순수함**(한 클래스만 있음). **클수록 불순함**. 분할 시 **감소량이 큰** 변수·기준으로 나눔.

### 10. 인공신경망 (Artificial Neural Network)

#### ● 퍼셉트론의 등장과 한계

* **단일층 퍼셉트론:** 인간 뉴런을 모사한 모델
* **한계:** **XOR 문제** 해결 불가 (선형 분리만 가능)

**출력 계산:** $W_1 X_1 + W_2 X_2 + b$  
예: $(0.6)(0.5) + (-2.0)(0.3) + 1.0 = 0.7$  
* **가중치:** 퍼셉트론 간 연결 강도. **편향(b):** 상수항.

#### ● 다층 퍼셉트론 (MLP)

**구조:** 입력층과 출력층 사이에 **은닉층** 1개 이상 삽입.

| 항목 | 설명 |
| :--- | :--- |
| **(1) 활성함수** | **시그모이드** 등으로 **비선형** 문제 해결 |
| **(2) 특징** | **블랙박스 모델** — 해석 어려움 |
| **(3) 하이퍼파라미터** | 은닉층 수·노드 수는 **사용자가 직접 설정** |
| **(4) 은닉층 수** | 많으면 → 복잡한 문제 해결 가능, **과적합 위험↑** / 적으면 → **과소적합** 가능 |

#### ● 인공신경망 학습 방법

| 방법 | 설명 |
| :--- | :--- |
| **(1) 순전파 (Feedforward)** | 정보가 **입력 → 출력** 방향으로 전달 |
| **(2) 역전파 (Backpropagation)** | 가중치 수정으로 **손실함수 감소**. **합성함수의 곱(연쇄법칙)** 활용 |
| **(3) 경사하강법 (Gradient Descent)** | **편미분**으로 경사 내리막 방향 이동, 오차 최소화 |
| **(4) 기울기 소실 (Vanishing Gradient)** | 은닉층이 많을 때 **시그모이드** 사용 시, 역전파 기울기가 0에 가까워져 **학습이 잘 되지 않는** 문제 |

#### ● 활성함수와 손실함수

**은닉층 활성함수** (인공신경망의 선형성 극복):

| 함수 | 출력 범위 | 특징 |
| :--- | :--- | :--- |
| **시그모이드** | 0 ~ 1 | 확률값. 기울기 소실 위험 |
| **Tanh (하이퍼볼릭 탄젠트)** | -1 ~ 1 | 시그모이드보다 기울기 소실 **지연** |
| **ReLU** | max(0, x) | 기울기 소실 **극복** |

**출력층 활성함수:**

| 함수 | 용도 | 수식 |
| :--- | :--- | :--- |
| **시그모이드** | 이진 분류 | $\sigma = 1/(1+e^{-x})$ |
| **소프트맥스** | 다중 분류 (합=1) | $\sigma_{x_k} = e^{x_k} / \sum e^{x_i}$ |

**손실함수** (예측값과 실제값 차이 측정):

| 손실함수 | 용도 |
| :--- | :--- |
| **MSE (평균제곱오차)** | 회귀 |
| **이진 크로스엔트로피** | 이진 분류 |
| **크로스엔트로피** | 다중 분류 |

#### ● 인공신경망 과적합 방지

| 방법 | 설명 |
| :--- | :--- |
| **규제** | L1(라쏘), L2(릿지) |
| **드롭아웃** | 훈련 시 일부 퍼셉트론 **비활성화** |
| **조기종료** | 특정 시점에서 학습 **조기 중단** |
| **모델 복잡도 줄이기** | 은닉층 노드 수 감소 |
| **데이터 증강** | 기존 데이터 변환으로 **데이터량 증가** |
| **배치정규화** | 각 층 출력 **정규화** |

#### ● 규제(Regularization)란?

**정의:** 손실함수에 **패널티 항**을 더해 가중치가 과도하게 커지는 것을 억제 → **과적합 방지**.

| 구분 | L1 (라쏘) | L2 (릿지) |
| :--- | :--- | :--- |
| **패널티** | $\sum |w|$ | $\sum w^2$ |
| **효과** | 일부 가중치를 **0**으로 → 변수 선택 | 가중치를 **작게** 유지 |
| **특징** | 희소 모델 | 모든 가중치 골고루 감소 |

* **총 손실** = 손실 + $\lambda \cdot$ 패널티. $\lambda$: 규제 강도.
* **엘라스틱넷**: L1 + L2 동시 적용.

### 11. 서포트벡터머신 (SVM)

**정의:** **마진이 최대**가 되는 초평면을 찾아, 선형·비선형 **이진 분류** 및 **회귀**에 활용하는 모델.

#### ● 핵심 구성요소

| 구성요소 | 설명 |
| :--- | :--- |
| **(1) 하이퍼플레인(초평면)** | 데이터를 구분하는 **경계**. 가중치벡터와 편향으로 결정 |
| **(2) 서포트벡터** | 초평면에 **가장 가까운** 샘플들 |
| **(3) 마진** | 초평면과 서포트벡터 사이의 **거리** |
| **(4) 커널함수** | 저차원 데이터를 **고차원**으로 변환하는 함수 (비선형 분리 가능) |

#### ● 커널함수 (Kernel Function) ⚠️

> [!IMPORTANT] 핵심
> **커널함수** = 저차원 공간의 데이터를 **고차원**으로 매핑하여, **원래 공간에서는 선형 분리 불가능한** 문제를 고차원에서 **선형으로 분리** 가능하게 하는 함수. SVM이 비선형 문제를 해결하는 핵심 장치.

* **역할:** 2차원에서 겹쳐 보이는 두 클래스를 3차원 이상으로 올리면 초평면으로 나눌 수 있게 함.
* **커널 트릭:** 고차원에서의 내적을 **저차원 계산만으로** 구할 수 있어, 실제 고차원 변환 없이 비선형 분리 가능.
* **대표 커널:** 선형, RBF(방사기저), 다항(Polynomial), 시그모이드 등.

#### ● SVM 유형

| 유형 | 설명 |
| :--- | :--- |
| **하드마진분류** | 오류 **비허용** — 엄격한 분리 |
| **소프트마진분류** | 마진 내 **어느 정도 오류 허용** — 비선형·노이즈에 유연 |

### 12. 연관성분석 (Association Analysis)

**정의:** 항목들 간 **조건–결과** 패턴을 발견하는 기법. **장바구니 분석**이 대표 예.

#### ● 연관분석 특징

| 특징 | 설명 |
| :--- | :--- |
| **결과 형태** | 단순·명확 (**IF~THEN~** 규칙) |
| **비목적성** | 특정 목표변수 없이 패턴 탐색 (비지도) |
| **계산량** | 품목 수 증가 시 **기하급수적** 증가 |

#### ● 연관분석 지표 (Support, Confidence, Lift)

| 지표 | 수식 | 의미 |
| :--- | :--- | :--- |
| **지지도 (Support)** | $P(A \cap B) = N(A \cap B) / \text{전체}$ | A와 B가 **동시에** 포함된 거래 비율 |
| **신뢰도 (Confidence)** | $P(A \cap B) / P(A)$ | A 구매 시 B도 구매할 **조건부 확률** |
| **향상도 (Lift)** | $P(A \cap B) / (P(A) \cdot P(B))$ | A와 B의 **상관성** |

**향상도 해석:**

| 향상도 | 의미 |
| :--- | :--- |
| **> 1** | 양의 상관관계 — 함께 구매되는 경향 |
| **= 1** | 상관없음 |
| **< 1** | 음의 상관관계 — 함께 구매되지 않는 경향 |

#### ● 계산 예시: 맥주 → 치킨

거래 200건 중 맥주 100건, 치킨 90건, 맥주·치킨 동시 60건일 때:

* $P(\text{맥주}) = 100/200 = 0.5$, $P(\text{치킨}) = 90/200 = 0.45$
* **지지도** $P(\text{맥주} \cap \text{치킨}) = 60/200 = 0.3$
* **신뢰도** (맥주→치킨) $= 0.3 / 0.5 = 0.6$ (60%)
* **향상도** $= 0.3 / (0.5 \times 0.45) = 1.33$ → **양의 상관관계**

> [!NOTE] 순차패턴분석
> 연관분석에 **시간 개념**을 추가한 분석. 시계열적 순서를 고려한 패턴 발견.
