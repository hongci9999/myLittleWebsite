## 2. 공통 트렌드 (2024-2025)

- **미니멀 + 모션**: 깔끔한 레이아웃 + 정교한 애니메이션
- **스위스 프린트**: 오프셋 그리드, 넉넉한 여백, 강한 타이포그래피
- **콘텐츠 우선**: 기술 과시보다 작업·내용 전시
- **제한된 팔레트**: 중립 톤 + 1~2 액센트 컬러

## 3. 헤더 베스트 프랙티스 (NN/G, Cube Creative)

- **스티키 헤더**: 작게 유지해 콘텐츠 비율 확보
- **최소 요소**: 로고, 네비, CTA(필요 시)
- **시각적 대비**: 헤더와 콘텐츠 구분
- **모션**: 자연스럽고 반응적으로

## 4. 적용 방향 (myLittleWebsite)

- 자료 정리·포트폴리오·기술 학습 목적에 맞춘 **미니멀·프로페셔널** 톤
- 기존 테마(blue-orange, amber-cyan, dark-slate) 유지
- 헤더: 단일 행, 로고 + 네비 + 테마 토글, 높이 최소화
- 타이포그래피: 계층 명확, 여백 충분
