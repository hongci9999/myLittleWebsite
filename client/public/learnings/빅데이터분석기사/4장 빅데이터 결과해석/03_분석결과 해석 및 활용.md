# 분석결과 해석 및 활용 (Interpretation and Utilization of Analysis Results)

## 1. 박스플롯의 해석 (Interpretation of Box Plot)

> [!NOTE] 박스플롯
> 평균(Mean)은 표시되지 **않는다**. 사분위수(Q1, Q2, Q3)와 극단값을 보여준다.

### 두 집단(A, B) 비교 시 해석 포인트

| 항목 | 해석 |
| :--- | :--- |
| **분산 (Variance)** | B의 데이터가 A보다 **넓게 퍼져 있으면** → B의 분산이 A보다 크다 |
| **중앙값 (Median)** | B의 제2사분위(Q2) 위치가 더 위에 있으면 → B의 중앙값이 A보다 크다 |
| **이상치 (Outlier)** | A에 상자 밖(최대 수염·MAX 초과) 데이터가 있으면 → A에 **이상치가 있다** |
| **평균 (Mean)** | 박스플롯만으로는 A·B의 **평균은 알 수 없다** |

---

## 2. summary() 함수 결과의 해석 (R/Python)

### (1) 연속형 변수 예시: Age

```
Min: 0.17 | 1st Qu: 21.00 | Median: 27.00 | Mean: 30.27 | 3rd Qu: 39.00 | Max: 76.00 | NA's: 86
```

| 해석 | 내용 |
| :--- | :--- |
| **변수 유형** | Mean, Median 존재 → **수치형 변수** |
| **사분위수** | 25% 지점 = 21.00, 75% 지점 = 39.00 |
| **왜도 (Skewness)** | **Median(27) < Mean(30.27)** → 왜도 > 0 → **우측으로 치우침 (Right-skewed)** |
| **결측치** | **NA's: 86** → 결측 처리(대체·제거) 필요 |

> [!TIP] 왜도 판단
> Median < Mean → 우측 치우침 (오른쪽 꼬리 길음)

---

### (2) 범주형 변수 예시: Survived

```
0: 266 | 1: 152
```

| 해석 | 내용 |
| :--- | :--- |
| **변수 유형** | 각 범주의 **빈도** 표시 → **범주형 변수** |
| **클래스 불균형** | 0(266) vs 1(152) → **불균형 상태** |
| **대응 방안** | **0 다운샘플링** 또는 **1 오버샘플링** 적용 |

## 3. 히트맵 해석 (Correlation Heatmap Interpretation)

Pearson 상관계수 히트맵 예시 (mtcars: mpg, disp, hp, drat, wt, qsec)

### 색상 해석

| 색상 | 상관계수 | 의미 |
| :--- | :--- | :--- |
| **빨강** | 1에 가까움 | 강한 **양의 상관** |
| **흰색** | 0 근처 | 상관 없음 |
| **파랑** | -1에 가까움 | 강한 **음의 상관** |

---

### 해석 포인트

| 유형 | 예시 (mtcars) |
| :--- | :--- |
| **강한 음의 상관** | mpg(연비) ↔ disp, hp, wt → 연비가 높을수록 배기량·마력·무게는 **낮다** |
| **강한 양의 상관** | wt(무게) ↔ disp, hp → 무게가 클수록 배기량·마력이 **크다** |

> [!WARNING] 독립변수 간 상관이 높을 때
> 회귀분석에서 **다중공선성** 문제가 발생할 수 있다.

| 대응 방안 | 설명 |
| :--- | :--- |
| **PCA (주성분분석)** | 차원 축소로 공선성 있는 변수들을 **주성분으로 통합** |
| **변수 제거** | 상관이 높은 변수 중 **일부를 선택적으로 제거** |

## 4. 회귀분석의 분산분석(ANOVA)표

> [!NOTE] ANOVA 검정
> 3개 이상 집단의 평균 비교. 회귀분석에서는 **모형의 유의성** 검정에 사용.

### ANOVA 표 구조

| 요인 | 제곱합 | 자유도 | 제곱평균 | F비 |
| :--- | :--- | :--- | :--- | :--- |
| **회귀** | $SSR = \sum(\hat{Y} - \bar{Y})^2$ | $p$ | $MSR = \frac{SSR}{p}$ | $F = \frac{MSR}{MSE}$ |
| **잔차** | $SSE = \sum(Y - \hat{Y})^2$ | $n - p - 1$ | $MSE = \frac{SSE}{n-p-1}$ | |
| **총** | $SST = SSR + SSE$ | $n - 1$ | | |

* $n$: 총 데이터 수, $p$: 회귀계수 수

### 관련 공식

| 항목 | 공식 |
| :--- | :--- |
| **총 데이터 수** | (자유도) + 1 |
| **R² (결정계수)** | $R^2 = \frac{SSR}{SST}$ |
| **수정된 R²** | $Adjusted\ R^2 = 1 - \frac{(n-1) \cdot MSE}{SST}$ |

### F비 해석

* **F값이 크고 p-value < 0.05** → 회귀모형이 **통계적으로 유의**하다 (모형이 의미 있게 설명한다).

---

### 예시: 연비(mpg)를 무게(wt)·마력(hp)으로 설명하는 회귀

* 데이터: $n = 32$ (샘플 수), 독립변수 2개 → $p = 2$

| 요인 | 제곱합 | 자유도 | 제곱평균 | F비 |
| :--- | :--- | :--- | :--- | :--- |
| **회귀** | SSR = 700 | 2 | MSR = 700÷2 = **350** | F = 350÷10.3 ≈ **34.0** |
| **잔차** | SSE = 300 | 29 | MSE = 300÷29 ≈ **10.3** | |
| **총** | SST = 1000 | 31 | | |

#### 단계별 해석

| 단계 | 계산/해석 |
| :--- | :--- |
| **1) 자유도** | 회귀 df = $p$ = 2, 잔차 df = $n-p-1$ = 29, 총 df = $n-1$ = 31 |
| **2) 제곱평균** | MSR = SSR÷2 = 350, MSE = SSE÷29 ≈ 10.3 |
| **3) F비** | F = MSR÷MSE = 350÷10.3 ≈ 34.0 → **회귀의 설명력이 잔차(오차)보다 월등히 큼** |
| **4) R²** | R² = SSR÷SST = 700÷1000 = **0.70** → 독립변수가 Y 변동의 **70%**를 설명 |
| **5) p-value** | F=34.0일 때 p-value는 0.05보다 매우 작음 → **모형 유의** |

> [!TIP] 결론
> F가 크고 p < 0.05이므로, wt·hp로 mpg를 설명하는 회귀모형은 **통계적으로 유의**하며, R²=0.70으로 설명력이 괜찮다.

---

### 예시: R `summary(lm())` 출력 해석

**모형:** `height ~ age + no_siblings` (키를 나이·형제 수로 설명)

#### R 출력 요약

| 구분 | 항목 | 값 | 해석 |
| :--- | :--- | :--- | :--- |
| **종속변수(Y)** | height | | 키 |
| **독립변수(X)** | age, no_siblings | | 나이, 형제 수 |
| **계수** | (Intercept) | 64.96 | p < 0.05 → 유의 |
| | age | 0.635 | p < 0.05 → **유의** (***) |
| | no_siblings | -0.011 | p = 0.851 → **유의하지 않음** |
| **모형** | R-squared | 0.9888 | 변동의 **98.88%** 설명 |
| | Adjusted R-squared | 0.9863 | |
| | F-statistic | 397.7 (2, 9 df) | p = 1.66e-09 → **모형 유의** |

#### 해석 절차

| 단계 | 내용 |
| :--- | :--- |
| **1) 모형 유의성** | F의 p-value(1.66e-09) < 0.05 → **전체 모형 통계적으로 유의** |
| **2) 개별 변수 유의성** | age: 유의 / no_siblings: **유의하지 않음** (p=0.851) |
| **3) 설명력** | R² ≈ 0.99 → 데이터 변동의 **98% 이상** 설명 |
| **4) 표본 수** | 회귀 df 2 + 잔차 df 9 + 1 = **12개** 데이터 |
| **5) 회귀식** | $\hat{Y}_{height} = 0.635 \cdot age - 0.011 \cdot no\_siblings + 64.96$ |

> [!WARNING] 유의하지 않은 변수
> `no_siblings`는 p=0.851로 유의하지 않다. **해당 변수를 제외하고 회귀를 재실행**하는 방안을 검토할 수 있으며, 최종적으로는 연구자의 판단에 따른다.

## 5. 주성분 분석(PCA) 결과 해석

### (1) PCA 특징

| 항목 | 설명 |
| :--- | :--- |
| **정의** | **상관이 높은 변수들의 선형결합**으로 새 변수를 만들어 **차원 축소** |
| **제1주성분(PC1)** | 데이터에서 **분산이 가장 큰** 축 (고유값 기준) |
| **제2주성분(PC2)** | 제1주성분에 **직교**하면서 그다음으로 분산이 큰 축 |
| **주성분 개수 결정** | 보통 **누적 설명력 70~90%**를 기준으로 선택 |

---

### (2) R `prcomp()` 결과 해석

```r
result <- prcomp(data, center=T, scale.=T)
summary(result)
```

| 출력 | PC1 | PC2 | PC3 | … | 해석 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Standard deviation** | 2.11 | 1.09 | 0.72 | | 각 주성분의 표준편차 |
| **Proportion of Variance** | 0.637 | 0.171 | 0.074 | | 각 주성분이 설명하는 **분산 비율** |
| **Cumulative Proportion** | 0.637 | 0.808 | 0.882 | … 1.0 | **누적** 설명력 |

* PC1 분산 비율이 가장 크다 (0.637).
* **PC1 + PC2**로 전체 분산의 **약 80%** 설명 (Cumulative ≈ 0.808).

---

### (3) 스크리플롯 (Scree Plot)

| 항목 | 설명 |
| :--- | :--- |
| **정의** | 주성분 개수 선택을 위한 그래프. X축=주성분 번호, Y축=분산(또는 고유값) |
| **선택 기준** | 그래프가 **수평으로 꺾이기 직전**까지의 주성분 개수를 선택 |

#### 해석 예

* PC1→PC2에서 분산이 급감, PC3~PC4 구간에서 완만해짐
* **2개 주성분** 선택 (꺾임 직전)
* 3개를 선택하는 것도 가능 — 연구 목적·판단에 따라 결정

## 7. 분석결과 시각화 — 시공간 시각화 (Spatiotemporal Visualization)

### (1) 시간 시각화 (Time Visualization)

**시간에 따른 변화** 표현. 일반적으로 X축=시간, Y축=값.

| 차트 | 설명 | 용도 |
| :--- | :--- | :--- |
| **막대 그래프 (Bar Chart)** | 연도·기간별 막대 비교 | 시계열 값 비교 |
| **점 그래프 (Dot Plot)** | 시점별 점 표시 | 추이·흐름 강조 |
| **선 그래프 (Line Graph)** | 점들을 선으로 연결 | **추세·변동** 강조 |
| **경사 차트 (Slope Chart)** | **두 시점 간 변화**를 선으로 연결 | 전·후 비교 (예: 1월 vs 2월) |
| **영역 차트 (Area Chart)** | 누적 변화를 **쌓은 영역**으로 표시 | 시계열 누적·비율 |

---

### (2) 공간 시각화 (Spatial Visualization)

**지역·위치별 데이터 분포** 표현.

| 차트 | 정의 |
| :--- | :--- |
| **단계구분도 (Choropleth)** | 영역에 따라 **색의 채도**를 이용해 자료를 **계급으로 집단화** |
| **카토그램 (왜상통계지도)** | **데이터 값을 강조**하기 위해 **면적을 통한** 특정 지역 왜곡 |
| **카토그램 히트맵** | **같은 면적**의 배경을 쓰고, **색의 채도**로 값을 구분 |
| **등치선도 (Isopleth)** | **동일한 값**을 가진 지점을 선으로 연결 (등고선·등압선 형태) |
| **버블 플롯 맵** | 지도 위에 **버블의 크기**로 데이터 값을 표현 |

---

### (3) 관계 시각화 (Relationship Visualization)

**변수 간 관계**를 표현하는 차트.

| 차트 | 설명 | 유의점 |
| :--- | :--- | :--- |
| **산점도 (Scatter Plot)** | 양의·음의 **상관관계**를 한눈에 확인 | 두 변수 관계 탐색 |
| **버블 차트 (Bubble Chart)** | **3개 변수** — 세 번째는 **원의 크기**로 표현. 좁은 공간에 정보 압축 | 버블 겹침 시 해석 어려움, **면적 비교 시 시각 왜곡** 가능 |
| **덴드로그램 (Dendrogram)** | 군집 간 **계층적 관계**를 나무 모양으로 표시 | 군집분석 결과 시각화 |
| **히트맵 (Heatmap)** | 변수 간 **상관**을 **색상 스케일**로 표시 | 다중공선성 검토에 활용 |
| **상관도표 (Correlogram)** | 상관행렬을 **원 크기·색**(파랑=양, 빨강=음)으로 표현 | 상관강도·방향 시각화 |

---

### (4) 분포 시각화 (Distribution Visualization)

**데이터 분포·비율·계층** 표현.

| 차트 | 설명 | 유의점 |
| :--- | :--- | :--- |
| **히스토그램 (Histogram)** | 구간별 **빈도**를 막대로 표시. 분포 형태·왜도 확인 | 분포 모양, 이상치 파악 |
| **원형(파이) 차트 (Pie Chart)** | 원 전체=100%, **각 조각=범주별 비율** | 범주 수가 많으면 해석 어려움 |
| **박스플롯 (Box Plot)** | 사분위수·범위 표시. **중앙값 위치·수염 길이**로 왜도 파악 | 분포·이상치 시각화 |
| **모자이크 차트 (Mosaic Chart)** | **두 변수**에 걸친 비율을 **영역 크기**로 표현 | 음수 표현 불가, 인접하지 않은 영역 비교 어려움 |
| **트리맵 (TreeMap)** | **계층 구조**와 비율을 **중첩 사각형**으로 표현 | 모자이크와 유사한 한계, 비율·계층 시각화 |

> [!NOTE] 비율/관계 시각화
> 파이차트·모자이크·트리맵은 분석 목적에 따라 **관계·비율 시각화**로 분류하기도 한다.

### (5) 비교 시각화 (Comparison Visualization)

**집단·변수 간 비교**를 위한 차트.

| 차트 | 설명 |
| :--- | :--- |
| **히트맵 (Heatmap)** | 행·열 조합별 값을 **색상 강도**로 표현 (저차원 비교) |
| **버터플라이 차트 (Butterfly Chart)** | **두 집단**을 **좌·우 대칭** 막대로 비교 (예: 남/녀, 연령대별 인구) |
| **레이더/스타 차트** | **3개 이상 변수**의 균형·추세를 **직관적으로 비교**. 중심에서 방사형 축 |
| **체르노프 페이스 (Chernoff Faces)** | 데이터를 **인물 얼굴** 형태로 변환해 표현 (눈·입·머리 형태 = 변수값) |
| **평행 좌표 그래프** | **고차원 데이터** 시각화. 여러 세로축을 선으로 연결해 개체별 패턴 비교 |

### (6) 인포그래픽 (Infographics)

**정보 디자인** 스펙트럼: 정보형 메시지 ← → 설득형 메시지  
(데이터 시각화 → 정보 시각화 → **인포그래픽**)

#### 정의 및 특징

| 항목 | 설명 |
| :--- | :--- |
| **정의** | **Information + Graphic** — 목적에 맞게 정보를 시각적으로 표현 |
| **목표** | 패턴·발견보다 **설득형 메시지 전달** — 대중에게 전달 |
| **데이터** | **원 데이터를 직접 다루지 않음** — 가공·처리된 결과물 중심 |

#### 인포그래픽 유형 (7가지)

| 유형 | 설명 |
| :--- | :--- |
| **지도형** | 지도를 활용 |
| **도표형** | 표·그래프를 활용 |
| **타임라인형** | **시간 순서**로 정보 배열 |
| **스토리텔링형** | **이야기·서사** 구조로 구성 |
| **만화형** | 만화·일러스트 요소 활용 |
| **비교분석형** | **두 개 이상 대상**을 비교 |
| **컨셉 맵** | **내용 간 연결 관계** 표현 |

#### 인포그래픽의 원리

| 원리 | 내용 |
| :--- | :--- |
| **기본원리** | 단순성, 명확성, 중요성, 일관성, 가독성, 효과성, 터겟 독자 고려 |
| **오컴의 면도날** | **복잡한 설명보다 단순한 설명**이 선호되어야 함 |

## 8. 분석결과 활용 (Utilization of Analysis Results)

분석모형 구축 **이후** 수행하는 단계.

| 단계 | 설명 |
| :--- | :--- |
| **(1) 분석모형 전개** | 분석 결과를 **확장·적용** |
| **(2) 분석결과 활용 시나리오 개발** | **인사이트 도출** → 업무·비즈니스에 **반영** |
| **(3) 분석모형 모니터링** | **시점 변화**를 모형에 반영하도록 **지속 모니터링** |
| **(4) 분석모형 리모델링** | **신규 알고리즘·데이터**를 반영해 모형 **갱신** |
