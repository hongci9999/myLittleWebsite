# 교환회선
## 네트워크 토폴로지
버스형
메인 통신 회선에 다수의 단말기 연결
간단 주고
추가제거 용이
신뢰성 상승
회선 길이 제한, 기밀보장 어려움

성형 star
메인을 중심으로 다수의 ㅁ단말기
추가제거 용이, 중계기 수 가장 적음
메인 시스템 고장나면 다 마비

ring 환형
p2p 방식으로 연결
양방향 데이터 정송, 고장나도 다른쪽으로
추가제거 어려움 기밀보로 어려움 전송지연발생

tree 계층
인접한거끼리는 star
star 끼리는 bus
분산 처리 시스템 용이
특정 회선에 과도한 트래픽, 메인이 고장나면 마비

mesh 망형
모든 단말기 연결
연결성 높음 다량의 통신 가능
총 길이 가장 김
