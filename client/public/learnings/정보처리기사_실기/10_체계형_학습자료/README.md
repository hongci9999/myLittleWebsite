# 정보처리기사 실기 체계형 학습자료

날짜: 2026-04-14
태그: [정보처리기사, 실기, 학습자료, 체계화]

---

## 목적

- 실기 범위를 과목별로 빠르게 찾을 수 있게 재정리
- 기존 `SQL_프로그래밍` 폴더와 별개로, SQL 핵심 내용을 한 번 더 체계화
- 프로젝트의 기존 학습자료 탐색 구조와 호환되는 폴더/문서 구성 유지

## 구성

1. `00_개요` - 사용 가이드, 범위 설명
2. `01_프로그래밍_및_알고리즘` - 언어/알고리즘 핵심
3. `02_데이터베이스` - SQL, 정규화, 트랜잭션
4. `03_인프라_및_네트워크` - OS, 네트워크, 계산형 개념
5. `04_소프트웨어공학_및_테스트` - 테스트/결함/형상
6. `05_용어사전` - 단답형 키워드/약어
7. `06_SQL_프로그래밍_재정리` - 기존 SQL 파트와 별도 재정리본
