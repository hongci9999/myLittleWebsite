저장 - 메모리

명령어를 인출하는 단계 패치

## 명령어 주소 지정 방식
메모리 참조 횟수가 적을수록 접근 빠름

즉시 주소 지정
실제 데이터가 존재하여 즉시 참조
참조 0회

직접 주고 지정
연산 대상의 주소값을 저장
참조 1회
메모리 확장, 변경 어려움

간접 주소 지정
연산 대상 주소값이 저장되어있는 곳의 주고삾을 저장하는 방식
2회

주소 길이 상관없이 언제든 원하는 위치의 참조 가능

계산에 의한 주소 지정
주소값과 특정 레지스터의 값을 연산하여 실제 데이터가 있는 위치를 계산

## 레지스터
연산에 필요한 데이터 및 상태값을 일시적으로 저장
가격 대비 용량이 작고 속도는 가장 빠음

플립플롭 - 동기식
래치 - 비동기식

제어장치 cu
모든 장치들의 동작이나 연산을 제어
메모리 주소 레지스터 mar - 데이터의 위치를 기억하는 레지스터
메모리 버퍼 레지스터 mbr  데이터가 잠시 저장
프로그램 계수기 - 다음에 실행할 명열어들의 위치 보관
명령 레지스터 - 현재 실행 중인 명령어 보관
명령 해독기 - 명령 레지스터에 있는 명령을 해독하는 회호
부호기 - 해독괸 명령어를 신호로 생성하는 회로
번지 레지스터 - 번지를 기억하는 장치

연산장치
실제 연산을 수행하는 장치
가산기 - 값을 더하는
누산기 - 일시적 보관 
보수기 -0 뺄셈
데이터 레지스터 - 데이터를 기억
상태 - 이벤트 상태 기억
인덱스 - 주소 변경을 위해 사용
