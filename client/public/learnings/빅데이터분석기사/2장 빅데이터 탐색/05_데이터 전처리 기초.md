# 데이터 전처리 기초

> 이 문서는 **데이터의 종류**, 척도, 정제 방법, 스케일링, 인코딩 등 전처리 전반의 기초 개념을 포함합니다.

## 📌 핵심 요약
데이터 분석에 앞서 **데이터의 종류**를 이해하고, **데이터 정제**를 통해 품질을 높이는 과정입니다.

---

## 📝 상세 정리

### 1. 데이터 정제

#### ● 데이터의 종류

* **(1) 척도 기준 구분**
    * **질적 척도**
        * **명목척도:** 어느 집단에 속하는지 나타내는 자료 (예: 대학교, 성별)
        * **순서척도(서열척도):** 서열관계가 존재하는 자료 (예: 학년, 순위)
    * **양적 척도**
        * **등간척도(구간척도):** 구간 사이 간격이 의미가 있으며 **덧셈과 뺄셈만 가능** (예: 온도, 지수)
        * **비율척도:** 절대적 기준 0이 존재하고 **사칙연산 가능**, 가장 많이 수집되는 자료 (예: 무게, 나이)

* **(2) 관측 기준 구분**
    * **단변량 데이터:** 데이터의 특성이 하나인 데이터
    * **다변량 데이터:** 데이터의 특성이 두 개 이상인 데이터
    * **시계열 데이터:** 시간 순서에 따라 관측된 데이터 (=종단면적 데이터)
    * **참고:** 종단면적 데이터(여러 시점에 측정) ↔ 횡단면적 데이터(한 시점에 측정) / 패널데이터(종단+횡단)

#### ● 데이터 정제 방법

* **(1) 집계 (Aggregation):** 데이터를 요약 (합계, 평균, 분산, 개수, 최대/최소)
* **(2) 일반화 (Generalization):** 데이터의 일반적인 특성 추출
* **(3) 정규화 (Normalization):** 데이터를 정해진 구간으로 조정하여 상대적 차이 제거
* **(4) 스무딩(=평활화, Smoothing):** 노이즈(잡음)를 제거하여 추세를 부드럽게 만듦

---

### 2. 변수 변환 (Variable Transformation)

#### ● 수치형 변수 변환

* **(1) 표준화 (Z-Score):** 평균 0, 표준편차 1로 변환하여 표준정규분포 $N(0,1)$에 근사.
    * 공식: $Z = \frac{X - \mu}{\sigma}$
* **(2) 정규화 (Min-Max Scaling):** 데이터를 0~1 구간으로 변환.
    * 공식: $X_{new} = \frac{X - \min(X)}{\max(X) - \min(X)}$
* **(3) 로그 변환 (Log Transformation):** $X_{new} = \log X$. 한쪽(좌측)으로 치우쳐 있을 시 활용.
* **(4) 그 외:** 지수변환, 제곱근 변환, **Box-Cox 변환**(양수 데이터의 비대칭 분포 시 활용)

#### ● 범주형 변수 변환

* **(1) 레이블 인코딩 (Label Encoding):** 범주형 데이터를 정수로 변환. 서열 관계를 부여하고자 할 때 사용.
    * **주의:** 본래 서열이 없는 범주에 적용 시 **서열 부여 문제**가 발생할 수 있음.
    * 예: 오렌지→0, 바나나→1, 포도→2
* **(2) 원-핫 인코딩 (One-Hot Encoding):** 고유 값에 해당하는 컬럼만 1로 표시하고 나머지는 0으로 표시.
    * 많은 메모리 공간을 할당하지만, **각 범주 간 독립적**이라 서열 부여 문제가 없음.
    * 예: 오렌지→[1,0,0], 바나나→[0,1,0], 포도→[0,0,1]
* **(3) 타깃 인코딩 (Target Encoding):** 타깃 변수의 **평균값**으로 범주를 변환.

---

### 3. 불균형 데이터 처리 (Imbalanced Data Processing)

클래스 분포가 한쪽으로 치우친 데이터(예: 암환자 50 vs 정상 5)에서 소수 클래스를 과소평가하지 않도록 균형을 맞추는 기법입니다.

* **(1) 가중치 균형 적용 (Weighted Balance):** 불균형 데이터에 **가중치**를 부여하여 클래스별 영향력을 조정.
* **(2) 과소표집 (UnderSampling):** **다수 클래스**의 일부만 선택하여 소수 클래스와 비율을 맞춤.
    * 방법: 랜덤 추출법, 계통 추출법, 집락 추출법, 층화 추출법
* **(3) 과대표집 (OverSampling):** **소수 클래스**를 복사하거나 유사한 데이터를 생성하여 비율을 맞춤.
    * 방법: SMOTE, ADASYN, ROS(Random Over-Sampling)
