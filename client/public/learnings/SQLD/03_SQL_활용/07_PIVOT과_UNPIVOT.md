# PIVOT · UNPIVOT

## 핵심 요약

**PIVOT**: **Long(세로)** → **Wide(가로)** — 행이던 **과목** 값을 **열**로. **집계함수 필수**(예: `MAX`). **UNPIVOT**: **Wide → Long** — 열 이름을 **행(과목)** 으로. `FOR 열 IN (...)` 에서 **값↔열 이름** 매핑.

## 왜 중요한가

- 보고서·엑셀형 **가로 표**와 DB **정규형 세로 표** 변환이 필요하다.
- **PIVOT = MAX + FOR IN**, **UNPIVOT = FOR IN** 역변환 — 문법 **대칭**이 시험 포인트.
- Oracle·SQL Server 등 **DBMS 확장** 문법(표준 SQL과 별도).

> 집계: [05_SQL_함수_문자함수](../02_SQL_기본/05_SQL_함수_문자함수.md) §6

## 1. Long 데이터 vs Wide 데이터

| 형태 | 구조 | 예 |
|------|------|-----|
| **Long**(세로·좁음) | `이름`, `키(과목)`, `값(점수)` — **한 과목당 1행** | 수학·과학이 **각각 행** |
| **Wide**(가로·넓음) | `이름`, `수학`, `과학` — **과목이 열** | **한 사람 1행** |

```
Long                    PIVOT              Wide
id key val    ──────►    id  x  y  z
1  x   a                 1  a  c  e
1  y   c                 2  b  d  f
1  z   e
2  x   b      UNPIVOT ◄────── (역방향)
2  y   d
2  z   f
```

## 2. PIVOT 절

| 항목 | 내용 |
|------|------|
| **역할** | 특정 열의 **값**을 **새 열 헤더**로 펼침 |
| **방향** | **Long → Wide** |
| **집계** | 여러 행 → 1행이므로 **MAX**, **SUM** 등 **필수** |

### \<성적\> (Long)

| 이름 | 과목 | 점수 |
|------|------|------|
| 철수 | 수학 | 90 |
| 철수 | 과학 | 85 |
| 영희 | 수학 | 88 |
| 영희 | 과학 | 92 |

```sql
SELECT *
  FROM (
        SELECT 이름, 과목, 점수
          FROM 성적
       )
 PIVOT (
       MAX(점수)
       FOR 과목 IN ('수학' AS 수학, '과학' AS 과학)
      );
```

### 결과 (Wide)

| 이름 | 수학 | 과학 |
|------|------|------|
| 철수 | 90 | 85 |
| 영희 | 88 | 92 |

| 구문 | 역할 |
|------|------|
| **MAX(점수)** | 같은 이름·과목에 여러 행이면 **하나로** 묶기 |
| **FOR 과목 IN (...)** | `과목` 열 값 **'수학'·'과학'** → **열 이름** |
| **AS 수학** | 결과 열 **헤더** (생략 가능) |

**왜 `MAX(점수)`인가?** `PIVOT`은 **집계 함수**가 **필수**다. (이름, 과목)당 **행이 1개**면 `MAX`·`MIN`·`SUM` 모두 **그 점수**와 같다. **중복 행**이 있으면 `MAX`=최고점, `MIN`=최저점, `SUM`=합계로 **의미가 달라진다**. 교재·시험 예제는 보통 **1행 1점**이라 `MAX`를 많이 쓴다.

> **서브쿼리**로 `PIVOT` **대상 SELECT** 를 감싼 뒤 `PIVOT` 절을 붙이는 형태가 많음.

## 3. UNPIVOT 절

| 항목 | 내용 |
|------|------|
| **역할** | **열**을 **행**으로 내림 |
| **방향** | **Wide → Long** |

### \<성적_피벗\> (Wide)

| 이름 | 수학 | 과학 |
|------|------|------|
| 철수 | 90 | 85 |
| 영희 | 88 | 92 |

```sql
SELECT *
  FROM (
        SELECT 이름, 수학, 과학
          FROM 성적_피벗
       )
UNPIVOT (
         점수 FOR 과목 IN (수학 AS '수학', 과학 AS '과학')
        );
```

### 결과 (Long)

| 이름 | 과목 | 점수 |
|------|------|------|
| 철수 | 수학 | 90 |
| 철수 | 과학 | 85 |
| 영희 | 수학 | 88 |
| 영희 | 과학 | 92 |

| 구문 | 역할 |
|------|------|
| **점수** | **값**이 들어갈 **새 열** |
| **FOR 과목 IN (수학 AS '수학', …)** | **기존 열** → `과목` **행 값** |

---

## 4. PIVOT vs UNPIVOT

| 구분 | PIVOT | UNPIVOT |
|------|-------|---------|
| **방향** | Long → Wide | Wide → Long |
| **집계** | **필요** (`MAX` 등) | **없음** |
| **FOR … IN** | 행 값 → **열** | **열** → 행 값 |
| **입력** | 이름·과목·점수 | 이름·수학·과학 |

---

## 5. 시험 포인트 / 함정

| 구분 | 내용 |
|------|------|
| PIVOT | **세로→가로**, **집계함수 필수** |
| UNPIVOT | **가로→세로** |
| FOR IN | **피벗 열** 값 목록 **명시** |
| Long/Wide | Long=**속성·값 행**, Wide=**속성=열** |
| 함정 | PIVOT **집계 생략** → 오류 |
| 함정 | PIVOT·UNPIVOT **IN 목록** 순서·별칭 **반대** 헷갈림 |
