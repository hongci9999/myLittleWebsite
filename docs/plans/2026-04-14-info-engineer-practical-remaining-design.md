# 정보처리기사 실기 남은 단원 재설계

**날짜:** 2026-04-14  
**관련:** `client/public/learnings/정보처리기사_실기/10_체계형_학습자료`

**Goal:** 남아 있는 `인프라 및 네트워크`, `소프트웨어공학 및 테스트`, `용어사전`, `SQL 프로그래밍 재정리` 단원을 실기 시험용 강의 교안 수준으로 재작성한다.

---

## 1) 배경

- 현재 남은 단원은 모두 1~2개 수준의 요약 문서만 있다.
- 개념 분류는 맞지만, 실제 학습 기준 문서로 사용하기에는 깊이와 세분화가 부족하다.
- 앞서 개편한 프로그래밍/알고리즘, 데이터베이스 단원과 품질 격차가 크다.

---

## 2) 목표 품질

- 남은 단원도 동일한 형식의 강의노트형 문서로 통일한다.
- 각 문서마다 개념, 비교표, 예제, 함정, 시험장에서 푸는 순서를 포함한다.
- SQL 재정리 파트는 기존 상세 SQL 폴더와는 별도로 "시험 직전 압축 정리본" 역할을 강화한다.

## 5) 품질 기준

- 운영체제/네트워크는 계산형과 용어형을 모두 커버한다.
- 소프트웨어공학은 테스트 기법 비교와 프로세스 흐름을 명확히 정리한다.
- 용어사전은 분야별로 나눠 탐색성과 암기 편의성을 높인다.
- SQL 재정리는 시험 직전 빠르게 보는 압축본이지만, 함정 설명은 충분히 포함한다.
