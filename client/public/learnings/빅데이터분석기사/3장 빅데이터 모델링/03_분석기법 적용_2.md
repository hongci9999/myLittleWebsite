# 분석기법 적용 ② (Application of Analysis Techniques)

> [!NOTE] 관련 문서
> 회귀분석·로지스틱·의사결정나무·SVM은 [02_분석기법 적용.md](./02_분석기법%20적용.md) 참조.

## 📌 핵심 요약
**인공신경망**은 은닉층으로 비선형 문제를 해결하지만 **기울기 소실**·**블랙박스**가 이슈입니다. **연관성분석**은 IF~THEN 규칙으로 항목 간 패턴을 발견하는 비지도 기법입니다. (딥러닝 DNN·CNN은 [05_딥러닝 분석.md](./05_딥러닝%20분석.md) 참조) **군집분석**은 거리·유사성 기준으로 군집을 나누며, K-means(거리 기반)와 DBSCAN(밀도 기반)이 대표적입니다.

---

## 📝 상세 정리

### 1. 인공신경망 (Artificial Neural Network)

#### ● 퍼셉트론의 등장과 한계

* **단일층 퍼셉트론:** 인간 뉴런을 모사한 모델
* **한계:** **XOR 문제** 해결 불가 (선형 분리만 가능)

**출력 계산:** $W_1 X_1 + W_2 X_2 + b$  
예: $(0.6)(0.5) + (-2.0)(0.3) + 1.0 = 0.7$  
* **가중치:** 퍼셉트론 간 연결 강도. **편향(b):** 상수항.

#### ● 다층 퍼셉트론 (MLP)

**구조:** 입력층과 출력층 사이에 **은닉층** 1개 이상 삽입.

| 항목 | 설명 |
| :--- | :--- |
| **(1) 활성함수** | **시그모이드** 등으로 **비선형** 문제 해결 |
| **(2) 특징** | **블랙박스 모델** — 해석 어려움 |
| **(3) 하이퍼파라미터** | 은닉층 수·노드 수는 **사용자가 직접 설정** |
| **(4) 은닉층 수** | 많으면 → 복잡한 문제 해결 가능, **과적합 위험↑** / 적으면 → **과소적합** 가능 |

#### ● 인공신경망 학습 방법

| 방법 | 설명 |
| :--- | :--- |
| **(1) 순전파 (Feedforward)** | 정보가 **입력 → 출력** 방향으로 전달 |
| **(2) 역전파 (Backpropagation)** | 가중치 수정으로 **손실함수 감소**. **합성함수의 곱(연쇄법칙)** 활용 |
| **(3) 경사하강법 (Gradient Descent)** | **편미분**으로 경사 내리막 방향 이동, 오차 최소화 |
| **(4) 기울기 소실 (Vanishing Gradient)** | 은닉층이 많을 때 **시그모이드** 사용 시, 역전파 기울기가 0에 가까워져 **학습이 잘 되지 않는** 문제 |

#### ● 활성함수와 손실함수

**은닉층 활성함수** (인공신경망의 선형성 극복):

| 함수 | 출력 범위 | 특징 |
| :--- | :--- | :--- |
| **시그모이드** | 0 ~ 1 | 확률값. 기울기 소실 위험 |
| **Tanh (하이퍼볼릭 탄젠트)** | -1 ~ 1 | 시그모이드보다 기울기 소실 **지연** |
| **ReLU** | max(0, x) | 기울기 소실 **극복** |

**출력층 활성함수:**

| 함수 | 용도 | 수식 |
| :--- | :--- | :--- |
| **시그모이드** | 이진 분류 | $\sigma = 1/(1+e^{-x})$ |
| **소프트맥스** | 다중 분류 (합=1) | $\sigma_{x_k} = e^{x_k} / \sum e^{x_i}$ |

**손실함수** (예측값과 실제값 차이 측정):

| 손실함수 | 용도 |
| :--- | :--- |
| **MSE (평균제곱오차)** | 회귀 |
| **이진 크로스엔트로피** | 이진 분류 |
| **크로스엔트로피** | 다중 분류 |

#### ● 인공신경망 과적합 방지

| 방법 | 설명 |
| :--- | :--- |
| **규제** | L1(라쏘), L2(릿지) |
| **드롭아웃** | 훈련 시 일부 퍼셉트론 **비활성화** |
| **조기종료** | 특정 시점에서 학습 **조기 중단** |
| **모델 복잡도 줄이기** | 은닉층 노드 수 감소 |
| **데이터 증강** | 기존 데이터 변환으로 **데이터량 증가** |
| **배치정규화** | 각 층 출력 **정규화** |

#### ● 규제(Regularization)란?

**정의:** 손실함수에 **패널티 항**을 더해 가중치가 과도하게 커지는 것을 억제 → **과적합 방지**.

| 구분 | L1 (라쏘) | L2 (릿지) |
| :--- | :--- | :--- |
| **패널티** | $\sum |w|$ | $\sum w^2$ |
| **효과** | 일부 가중치를 **0**으로 → 변수 선택 | 가중치를 **작게** 유지 |
| **특징** | 희소 모델 | 모든 가중치 골고루 감소 |

* **총 손실** = 손실 + $\lambda \cdot$ 패널티. $\lambda$: 규제 강도.
* **엘라스틱넷**: L1 + L2 동시 적용.

### 2. 연관성분석 (Association Analysis)

**정의:** 항목들 간 **조건–결과** 패턴을 발견하는 기법. **장바구니 분석**이 대표 예.

#### ● 연관분석 특징

| 특징 | 설명 |
| :--- | :--- |
| **결과 형태** | 단순·명확 (**IF~THEN~** 규칙) |
| **비목적성** | 특정 목표변수 없이 패턴 탐색 (비지도) |
| **계산량** | 품목 수 증가 시 **기하급수적** 증가 |

#### ● 연관분석 지표 (Support, Confidence, Lift)

| 지표 | 수식 | 의미 |
| :--- | :--- | :--- |
| **지지도 (Support)** | $P(A \cap B) = N(A \cap B) / \text{전체}$ | A와 B가 **동시에** 포함된 거래 비율 |
| **신뢰도 (Confidence)** | $P(A \cap B) / P(A)$ | A 구매 시 B도 구매할 **조건부 확률** |
| **향상도 (Lift)** | $P(A \cap B) / (P(A) \cdot P(B))$ | A와 B의 **상관성** |

**향상도 해석:**

| 향상도 | 의미 |
| :--- | :--- |
| **> 1** | 양의 상관관계 — 함께 구매되는 경향 |
| **= 1** | 상관없음 |
| **< 1** | 음의 상관관계 — 함께 구매되지 않는 경향 |

#### ● 계산 예시: 맥주 → 치킨

거래 200건 중 맥주 100건, 치킨 90건, 맥주·치킨 동시 60건일 때:

* $P(\text{맥주}) = 100/200 = 0.5$, $P(\text{치킨}) = 90/200 = 0.45$
* **지지도** $P(\text{맥주} \cap \text{치킨}) = 60/200 = 0.3$
* **신뢰도** (맥주→치킨) $= 0.3 / 0.5 = 0.6$ (60%)
* **향상도** $= 0.3 / (0.5 \times 0.45) = 1.33$ → **양의 상관관계**

> [!NOTE] 순차패턴분석
> 연관분석에 **시간 개념**을 추가한 분석. 시계열적 순서를 고려한 패턴 발견.

#### ● 빈발항목 도출 알고리즘

| 알고리즘 | 정의 | 특징 |
| :--- | :--- | :--- |
| **(1) Apriori** | **최소 지지도**를 만족하는 빈발 항목을 추출해 **연산량**을 줄이는 기법 | 빈도수 탐색 → 최소 지지도 확인 → 후보 집합 생성 → 반복 탐색 → 연관규칙 도출 |
| **(2) FP-Growth** | Apriori의 **반복 과정**을 **트리 기반**으로 대체해 연산 효율 향상 | 후보 생성 없이 FP-Tree로 빈발 패턴 탐색. Apriori보다 **빠름** |

### 3. 군집분석 (Cluster Analysis)

**정의:** **비지도 학습**(Unsupervised Learning)으로, 데이터들 간의 **거리**나 **유사성**을 기준으로 군집을 나누는 분석 기법.

#### ● 군집분석 방법

| 방법 | 설명 |
| :--- | :--- |
| **(1) 계층적 방법 (Hierarchical Method)** | 데이터 간 유사성을 기준으로 군집을 형성하는 방식 |
| **(2) 비계층적 방법** | 계층적 방법 외의 군집분석 기법 (예: K-means 등) |

#### ● 덴드로그램 (Dendrogram)

계층적 군집분석의 결과를 시각화한 그래프.

* **Y축:** 거리 — 아래에서 위로 올라가며 군집이 합쳐지는 과정 표현
* **예시:** 거리를 15에서 나누면 3개 클러스터, 25에서 나누면 2개 클러스터로 분할 가능

#### ● 군집 간 거리 측정 방식 (계층적 군집분석)

| 방법 | 설명 |
| :--- | :--- |
| **최단 연결법 (=단일 연결법)** | 군집 간 **가장 가까운** 데이터를 기준으로 거리 측정 |
| **최장 연결법 (=완전 연결법)** | 군집 간 **가장 먼** 데이터를 기준으로 거리 측정 |
| **평균 연결법** | 두 군집 내 **모든 데이터 쌍** 간의 거리 **평균** 사용 |
| **중심 연결법** | 두 군집의 **중심점(Centroid)** 사이의 거리 측정 |
| **와드 연결법** | 두 군집이 합쳐졌을 때 **편차 제곱합**이 최소가 되는 위치를 기준으로 병합 |

### 4. 거리·유사도 측정

#### ● 거리 측정 방식 (연속형 변수)

| 거리 | 설명 |
| :--- | :--- |
| **유클리디안 거리 (Euclidean)** | 두 점 사이의 **직선 거리** |
| **맨하튼 거리 (Manhattan)** | 각 변수들의 차이의 **단순 합** |
| **체비세프 거리 (Chebyshev)** | 변수 간 거리 차 중 **최댓값** |
| **민코우스키 거리 (Minkowski)** | 유클리디안·맨하튼 거리를 **일반화**한 거리 |
| **마할라노비스 거리 (Mahalanobis)** | 표준화 거리에서 변수의 **상관관계**를 고려한 거리 |
| **표준화 거리 (Standardized)** | 변수의 스케일을 맞춘 뒤 계산하는 거리 |

#### ● 민코우스키 거리 공식

$$d = \left[ \sum |x_i - y_i|^p \right]^{1/p}$$

* $p=1$ → 맨하튼 거리
* $p=2$ → 유클리디안 거리
* $p \to \infty$ → 체비세프 거리

> [!IMPORTANT] 마할라노비스 거리
> 표준화 거리와 달리 변수 간 **상관관계**를 반영. 변수들이 서로 상관되어 있을 때 더 적절한 거리 측정.

#### ● 유사도 측정 (범주형 변수)

| 유사도 | 공식 | 설명 |
| :--- | :--- | :--- |
| **자카드 유사도 (Jaccard)** | $J = \frac{|A \cap B|}{|A \cup B|}$ | 집합 A와 B의 **교집합 크기**를 **합집합 크기**로 나눈 값 (0~1) |
| **코사인 유사도 (Cosine)** | $\cos \theta = \frac{A \cdot B}{\|A\| \|B\|}$ | 두 벡터의 **내적**을 각 벡터 **크기의 곱**으로 나눈 값 (-1~1) |

### 5. K-평균 군집화 (K-means Clustering)

**비계층적** 군집화 방법. **거리 기반**으로 군집 형성.

| 구분 | 내용 |
| :--- | :--- |
| **특징** | **군집 수 K를 사전에 지정**해야 함. 중심점 변경 시 **소속 군집이 변할 수 있음**. 이상치·초기 중심점에 민감 |
| **과정** | 1) K 설정 (**Elbow Method**로 최적 K 탐색) 2) K개 초기 중심점 임의 설정 3) 각 데이터를 가장 가까운 중심점 군집에 할당 4) 할당된 데이터 평균으로 중심점 재설정 5) 중심점 수렴할 때까지 3~4 반복 |

**K-means 변형:**

| 변형 | 설명 |
| :--- | :--- |
| **K-modes** | 범주형 데이터 — 평균 대신 **최빈값(mode)** 활용 |
| **K-medoids** | 실제 데이터 중 하나를 대표값으로 선정 — **이상치에 강건(Robust)** |

#### ● K-means 계산 예시

**데이터:** A(1,2), B(2,1), C(4,5), D(5,4) — K=2

| 단계 | 내용 |
| :--- | :--- |
| **1. 초기 중심점** | $C_1=(1,2)$, $C_2=(5,4)$ 임의 설정 |
| **2. 할당** | A,B → $C_1$ (가까움), C,D → $C_2$ (가까움). 유클리디안 거리 기준 |
| **3. 중심점 재설정** | $C_1 = \frac{A+B}{2} = (1.5, 1.5)$, $C_2 = \frac{C+D}{2} = (4.5, 4.5)$ |
| **4. 재할당** | A,B → $C_1$, C,D → $C_2$ (변동 없음) → **수렴** |

> [!NOTE] 예시 요약
> 좌하단(A,B)과 우상단(C,D)이 각각 한 군집으로 수렴. 중심점이 더 이상 변하지 않으면 알고리즘 종료.

### 6. DBSCAN

**비계층적** 군집화 방법. **밀도 기반**으로 군집 형성.

| 구분 | 내용 |
| :--- | :--- |
| **특징** | **군집 수를 사전에 정하지 않아도 됨**. 노이즈·이상치에 강함 |
| **파라미터** | **Eps**(반경), **MinPts**(최소 포인트 수) 사전 설정 필요 |

### 7. 기타 비계층적 군집분석

| 방법 | 기반 | 설명 |
| :--- | :--- | :--- |
| **(1) 퍼지군집화 (Fuzzy Clustering)** | **확률** 기반 | 각 데이터가 특정 군집에 속할 **확률**을 계산하여 군집화 |
| **(2) EM알고리즘 (Expectation-Maximization)** | **분포** 기반 | **E단계**: 우도의 기댓값 계산 / **M단계**: 최대우도 추정 — 두 단계 반복 |
| **(3) SOM (자기조직화지도)** | **그래프** 기반 | **신경망**으로 차원 축소 후 지도 형태로 시각화하여 군집화. **완전연결** 구조, **순전파**만 사용. **경쟁 학습** — 각 노드가 승자 되기 위해 경쟁 |

> [!TIP] K-means vs DBSCAN
> **K-means**: 거리 기반, K 사전 지정, 이상치 민감  
> **DBSCAN**: 밀도 기반, K 불필요, 이상치에 강함
