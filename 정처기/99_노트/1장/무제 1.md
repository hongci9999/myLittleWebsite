요구사항 검증
잘못된 요구사항으로 설계된걸 수정하려면 비용 많이 듬

유효성,일관성,완결성, 형실성, 검증 가능성 등 파악

검증 기법
-검토
동료검토 - 다수의 동료들에게 직접 설명
워크스루 - 미리 사전검토 ㅎ후 짋은 회의
인스펙션 - 전문 검토 그룹이 상세히 분석

프로토타이필
시제품을 신속 간단히 개발후 요구사항 검증
즉각적인 피드백, 사전 식별 가능
반복되면 비용 증가, 사용성 과대평가

모델 검증
개발된 모델이 만족하는지 검증
질세 실행을 통해 검증이 아닌 모델의 구조 및 의사소통 경로 들을 검증(정적분석) 하는 것
