# 기출·모의 SQL 문제 정리 (연도별)

**저작권**: 공개된 기출·문제지를 **직접 옮기지 말고**, 본인이 풀고 난 뒤 **요약·오답 포인트**만 기록하는 용도로 사용하세요.

---

## 사용 방법

1. 시험장에서 가져온 **문제 번호·유형**(DDL / SELECT 결과 / 권한 / 조인 등)만 적습니다.
2. **내가 쓴 답**과 **정답 포인트**를 2~3줄로 적습니다.
3. 모르는 문법은 해당 노트로 링크합니다. (예: `[[03_Join_and_Subquery]]` — Obsidian에서)
