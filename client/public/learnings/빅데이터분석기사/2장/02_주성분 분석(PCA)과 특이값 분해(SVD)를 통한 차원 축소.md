# 주성분 분석(PCA)과 특이값 분해(SVD)를 통한 차원 축소

## 📌 핵심 요약
고차원 데이터의 변동성을 최대한 보존하면서 서로 상관관계가 없는 새로운 변수(주성분)로 차원을 축소하는 통계적 기법인 **PCA**와, 행렬을 특정한 구조로 분해하여 데이터의 본질적인 특징을 추출하는 **SVD**에 대해 다룹니다. 이를 통해 데이터 시각화, 잡음 제거, 분석 효율성 향상을 도모할 수 있습니다.

## 📝 상세 정리

### 1. 주성분 분석 (PCA, Principal Component Analysis)
고차원 데이터의 차원을 축소하는 대표적인 통계 기법으로, 데이터 내의 주요한 변동성을 설명하는 새로운 변수를 찾아내어 구조를 이해하고 시각화합니다.

#### 1.1 목적 (Objectives)
* **차원 축소 (Dimension Reduction):** 데이터의 차원을 줄여 분석의 효율성을 높입니다.
* **시각화 (Visualization):** 고차원 데이터를 2차원 또는 3차원으로 투영하여 패턴을 파악합니다.
* **잡음 제거 (Noise Reduction):** 주요한 정보(분산)만 유지하고 불필요한 중복성이나 잡음을 줄입니다.
* **변수 간 상관성 제거 (Multicollinearity Removal):** 서로 상관된 변수들을 비상관된 주성분으로 변환하여 분석의 안정성을 높입니다.

#### 1.2 수행 과정 (Process)
1.  **데이터 표준화 (Data Standardization):** 변수 간의 단위나 크기 차이에 의한 결과 왜곡을 방지하기 위해 평균을 $0$, 표준편차를 $1$로 변환합니다.
2.  **공분산 행렬 계산 (Covariance Matrix Calculation):** 데이터 내 변수들이 서로 어떻게 변화하는지(상관성)를 측정합니다.
3.  **고유값(Eigenvalue)과 고유벡터(Eigenvector) 계산:**
    * **고유값($\lambda$):** 주성분이 설명하는 분산의 크기를 나타냅니다.
    * **고유벡터($v$):** 분산의 방향을 나타내며, 이 벡터들이 새로운 주성분 축을 형성합니다.
4.  **주성분 선택 (Selecting Principal Components):** 고유값의 크기에 따라 내림차순으로 정렬 후, 상위 주성분을 선택합니다. 큰 고유값을 가진 주성분일수록 데이터의 변동성을 많이 설명합니다.
5.  **데이터 변환 (Data Transformation):** 원래 데이터를 선택한 주성분 공간으로 사영(Projection)하여 축소된 차원에서 분석을 수행합니다.

---

### 2. 특이값 분해 (SVD, Singular Value Decomposition)
데이터의 특성을 유지하면서 임의의 행렬을 세 개의 행렬의 곱으로 분해하는 기법입니다. 데이터 구조를 단순화하여 본질적인 이해를 돕습니다.

#### 2.1 SVD의 정의와 구성
SVD는 $m \times n$ 행렬 $A$를 다음과 같이 분해합니다.
$$A = U \Sigma V^T$$
* **$U$ (왼쪽 특이벡터, Left Singular Vectors):** $AA^T$를 고유값 분해해서 얻은 직교행렬로, 행 공간(사용자 등)의 특징을 나타냅니다.
* **$\Sigma$ (특이값 행렬, Singular Values):** 대각 행렬로, 대각 원소(특이값)는 데이터의 중요도(에너지)를 나타내며 내림차순으로 정렬됩니다.
* **$V^T$ (오른쪽 특이벡터, Right Singular Vectors):** $A^T A$를 고유값 분해해서 얻은 직교행렬의 전치행렬로, 열 공간(아이템/변수 등)의 특징을 나타냅니다.

> [!INFO] 💡 AI 보충 설명: SVD의 기하학적 의미
> SVD는 선형 변환 관점에서 행렬을 **회전(Rotation) → 스케일링(Scaling) → 다시 회전(Rotation)**하는 세 단계로 분해한 것입니다.
> 1.  $V^T$: 입력 데이터를 새로운 축으로 **회전**시킵니다.
> 2.  $\Sigma$: 각 축의 방향으로 데이터를 특이값 크기만큼 **확장 또는 축소(스케일링)**합니다.
> 3.  $U$: 다시 한번 **회전**시켜 최종 결과 공간으로 보냅니다.
> 이 과정을 통해 데이터에서 가장 중요한 '축'이 무엇인지 파악할 수 있게 됩니다.

#### 2.2 SVD의 응용 (Applications)
* **차원 축소 (Dimension Reduction):** 특이값 중 가장 큰 몇 개($k$개)만 남기고 나머지를 제거하여 차원을 줄입니다. 이는 **Truncated SVD**라고도 불리며 PCA와 밀접한 관련이 있습니다.
* **잡음 제거 (Noise Reduction):** 상대적으로 크기가 작은 특이값 성분을 제거함으로써 미세한 잡음을 걸러냅니다.
* **행렬 근사 (Matrix Approximation):** 특이값이 큰 상위 성분만으로 원본 행렬을 근사하여 데이터를 압축합니다.
* **추천 시스템 (Recommendation Systems):** 사용자-아이템 행렬에서 잠재적인 선호 패턴을 찾아내는 협업 필터링 알고리즘(예: Matrix Factorization)의 핵심으로 사용됩니다.

> [!INFO] 💡 PCA와 SVD의 관계
> PCA는 데이터의 공분산 행렬에 대해 고유값 분해를 수행하는 것과 같습니다. 수학적으로 **표준화된 데이터**에 대해 SVD를 수행하면 그 결과는 PCA와 동일한 주성분을 산출하게 됩니다. 실제 계산 구현 시에는 수치적 안정성을 위해 PCA 대신 SVD 알고리즘을 사용하는 경우가 많습니다.
