# C언어 제어문 및 함수 활용

## 📌 핵심 요약
프로그램의 흐름을 제어하는 **선택 제어문([[if]], [[switch]])**과 특정 코드를 반복 수행하는 **반복문([[for]], [[while]])**, 그리고 코드의 재사용성을 높이는 **함수([[Function]])**와 **표준 라이브러리**의 활용법을 다룹니다. 특히 반복문의 제어 키워드와 함수 간 데이터 전달 방식인 **[[Call by Value]]**를 상세히 정리합니다.

---

## 📝 상세 정리

## 1. 선택 제어문 (Selection Control Statement)

### 1.1 [[if]] ~ [[else]]
- 조건식의 결과가 참(`True`)인 경우 블록 내의 코드를 실행합니다.
- **삼항 연산자:** `if else`문을 한 줄로 간결하게 표현한 연산자입니다.
> [!INFO] 💡 AI 보충 설명: 삼항 연산자 예시
> **형식:** `(조건식) ? 참일_때_값 : 거짓일_때_값;`
> ```c
> int a = 10, b = 20;
> int max = (a > b) ? a : b; // a가 b보다 크면 a를, 아니면 b를 max에 저장
> ```

### 1.2 [[switch]] ~ [[case]]
- **switch:** 단일 값과 비교하여 일치하는 `case` 라벨로 이동합니다.
- **동작:** 데이터가 일치하면 해당 영역부터 아래의 모든 코드를 함께 실행합니다. (Fall-through 현상)
- **[[break]]:** 현재 실행 중인 블록을 탈출합니다. 일치하는 `case`만 실행하고 싶을 때 반드시 사용해야 합니다.
- **[[default]]:** 일치하는 `case`가 없을 때 실행되는 영역으로, 보통 가장 아래에 위치합니다.

## 2. 반복문 (Iteration Statement)

### 2.1 [[for]]문
- **구조:** `for(초기식; 조건식; 증감식) { ... }`
- 정해진 횟수만큼 반복할 때 주로 사용합니다.
- **[[continue]]:** 반복문 내에서 `continue` 아래의 코드를 무시하고 즉시 다음 회차(증감식 단계)로 이동합니다.

### 2.2 조건 제한 반복문
- **[[while]]:** 조건식을 먼저 확인하고, 참인 동안 블록을 무한 반복합니다.
- **[[do-while]]:** 코드를 **최소 1회 실행**한 후 조건식을 확인합니다.
> [!INFO] 💡 AI 보충 설명: do-while 예시
> ```c
> int i = 10;
> do {
>     printf("%d", i); // i가 10이므로 조건에 맞지 않지만, 일단 출력됨
> } while(i < 5); // 출력 결과: 10
> ```
- **무한 루프([[Infinite Loop]]):**
    - **C언어:** `while(1)` (0이 아닌 모든 숫자는 참)
    - **Java:** `while(true)`
    - **Python:** `while True:`

## 3. 함수 (Function) 및 라이브러리

### 3.1 함수의 특징
- 함수를 호출하면 호출한 프로그램(Main)은 잠시 멈추고 함수 내부 코드를 실행합니다.
- **정의와 호출:** 보통 `main` 함수 밖에서 정의하며, 반환값이 없을 경우 자료형을 **[[void]]**로 선언합니다.
- **용어 구분:**
    - **인수(Argument):** 호출하는 곳에서 함수로 넘겨주는 실제 값.
    - **매개변수(Parameter):** 외부에서 넘어온 인수를 함수 내부에서 받기 위해 선언된 변수.

### 3.2 표준 라이브러리 ([[Standard Library]])
미리 개발된 함수들을 기능별로 묶어 제공하며, `#include <파일명.h>`를 통해 불러옵니다.
- **stdio.h:** 표준 입출력 (printf, scanf 등)
- **math.h:** 수학 관련 (sqrt: 제곱근, pow: 거듭제곱, abs: 절대값)
- **string.h:** 문자열 처리 (strlen: 길이, strcpy: 복사, strcmp: 비교)
- **stdlib.h:** 자료형 변환, 메모리 할당, 의사 난수 등

### 3.3 값에 의한 전달 ([[Call by Value]])
- 함수 호출 시 인수의 **값 자체를 복사**하여 전달하는 방식입니다.
- 함수 내부에서 매개변수의 값을 변경해도 **호출한 곳의 원본 변수 값은 변하지 않습니다.**

> [!INFO] 💡 AI 보충 설명: Call by Value 예시
> ```c
> void swap(int x, int y) {
>     int temp = x;
>     x = y;
>     y = temp; // x, y 값은 바뀌지만 원본 a, b는 유지됨
> }
> int main() {
>     int a = 10, b = 20;
>     swap(a, b);
>     printf("%d %d", a, b); // 출력: 10 20
> }
> ```

## 🎯 빈출 키워드
- **@([[break]] / [[continue]])**: 반복 및 선택 제어문에서 흐름을 끊거나 건너뛰는 제어 키워드.
- **@([[do-while]])**: 조건 검사 전 최소 1회 실행을 보장하는 반복문.
- **@([[Standard Library]])**: stdio.h, math.h 등 주요 헤더 파일의 역할.
- **@([[Call by Value]])**: 인수의 값을 복사하여 전달하는 방식의 특징(원본 불변).
- **@([[void]])**: 리턴값이 없는 함수의 반환 타입.
