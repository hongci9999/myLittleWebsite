# 통계기법 이해 (Understanding Statistical Techniques)

## 📌 핵심 요약
**기술통계**는 수집된 데이터를 요약·설명하는 통계이며, **표본추출**은 모집단의 일부를 선택하여 분석하는 방법입니다. 전수조사와 표본조사의 차이, 확률적 표본추출 방법을 이해합니다.

---

## 📝 상세 정리

### 1. 기술통계 (Descriptive Statistics)

#### ● 전수조사와 표본조사

| 구분 | 설명 |
| :--- | :--- |
| **전수조사 (Census)** | 전체를 다 조사. 시간과 비용이 많이 소모됨. |
| **표본조사 (Sample Survey)** | 일부만 추출하여 모집단을 추론·분석. |

#### ● 확률적 표본 추출 방법 (Probabilistic Sampling Methods)

* **(1) 랜덤 추출법 (Random Sampling):** 무작위로 표본 추출.
* **(2) 계통 추출법 (Systematic Sampling):** 번호를 부여한 뒤 **일정 간격**으로 추출.
* **(3) 집락 추출법 (=군집 추출법, Cluster Sampling):**
    * 여러 **군집**으로 나눈 뒤 군집을 선택하여 랜덤 추출.
    * **군집 내 이질적**, **군집 간 동질적** 특징.
* **(4) 층화 추출법 (Stratified Sampling):**
    * **층 내 동질적**, **층 간 이질적** 특징.
    * 같은 비율로 추출 시 → **비례 층화 추출법**.
#### ● 집락 vs 층화 구분

| 구분 | 집락 추출법 (Cluster) | 층화 추출법 (Stratified) |
| :--- | :--- | :--- |
| **군집 내** | **이질적** (다양함) | **동질적** (비슷함) |
| **군집 간** | **동질적** (비슷함) | **이질적** (다양함) |
| **추출 방식** | 군집 단위로 선택 후, 선택된 군집 내에서 추출 | 각 층(집단)에서 비례하여 추출 |
| **예시** | 전국 학교 중 일부 지역 학교만 선택 → 그 학교 학생 전원 조사 | 남/여, 연령대별로 나눈 뒤 각 층에서 비율에 맞게 추출 |

> [!TIP] 암기법
> **집락:** 군집이 서로 비슷(동질) → "지역·학교"처럼 자연스럽게 묶인 단위. 군집 안은 다양한 구성.
> **층화:** 층이 서로 다름(이질) → "성별·연령"처럼 목적에 맞게 나눈 그룹. 층 안은 비슷한 구성.

* **(5) 복원·비복원 추출:**
    * **복원 추출:** 추출된 데이터를 다시 포함시켜 표본 추출.
    * **비복원 추출:** 추출된 데이터는 제외하고 표본 추출.

> [!INFO] 💡 깁스 샘플링 (Gibbs Sampling)
> 복잡한 확률 분포의 경우, **MCMC** 기반 조건부 확률을 이용해 샘플링하는 **깁스 샘플링** 방법도 활용됩니다.

### 2. 확률분포 (Probability Distribution)

#### ● 기초 확률 용어 (Basic Probability Terms)

* **(1) 확률 (Probability):** 통계적 현상이 일어날 **확실성의 정도**를 나타내는 척도. 수학적 확률·통계적 확률로 구분.
* **(2) 사건 (Event):** 통계적 시행의 결과로 나타나는 **표본공간의 부분집합**.
* **(3) 표본공간 (Sample Space):** 통계적 시행에서 발생 가능한 **모든 결과의 집합**.
    * 예: 동전 2개 던질 때 $S = \{(앞,앞), (앞,뒤), (뒤,앞), (뒤,뒤)\}$
* **(4) 확률변수 (Random Variable):** 표본공간의 각 원소를 **특정 값(확률)**에 대응시키는 함수.
    * 예: 주사위를 던질 때 나오는 눈(1~6)이 확률변수 $X$의 가능한 값.
* **(5) 조건부 확률 (Conditional Probability):** 사건 $B$가 일어났을 때 사건 $A$가 일어날 확률.
    * $P(A|B) = \frac{P(A \cap B)}{P(B)}$
    * 예: 백신을 맞았을 때 감염될 확률.
* **(6) 독립사건 (Independent Events):** 사건 $A$, $B$가 서로 영향을 주지 않음.
    * $P(A|B) = P(A)$ 또는 $P(A \cap B) = P(A) \cdot P(B)$
    * 예: 주사위 A에서 3이 나왔을 때, 주사위 B에서 3이 나올 확률 (서로 무관).
* **(7) 배반사건 (Mutually Exclusive Events):** 사건 $A$, $B$가 **동시에 일어날 수 없음**.
    * $A \cap B = \emptyset$ → $P(A \cap B) = 0$
    * 예: 동전 1개를 던질 때 앞면과 뒷면이 동시에 나오는 경우.
* **(8) 베이즈 정리 (Bayes' Theorem):** 사전확률과 사후확률의 관계를 나타내는 정리.
    * $P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}$

#### ● 확률분포의 기댓값 (Expected Value)

확률변수 $X$의 확률분포 $f(x)$에 대한 기댓값 $E(X)$:

* **이산적 확률변수:** $E(X) = \sum x \cdot f(x)$
* **연속적 확률변수:** $E(X) = \int x \cdot f(x) \, dx$

#### ● 이산 확률분포 (Discrete Probability Distribution)

확률분포는 **확률변수의 개별 값들이 가지는 확률 값의 분포**입니다. 이산형은 값을 셀 수 있으며 **확률질량함수(PMF)**로 표현합니다.

| 분포 | 정의 | 예시 |
| :--- | :--- | :--- |
| **베르누이 (Bernoulli)** | 결과가 두 가지 중 하나로 나타나는 1회 시행 | 동전 던지기, 시험 합격/불합격 |
| **이항 (Binomial)** | $n$번 베르누이시행 중 $k$번 성공할 확률. $\mu=np$, $\sigma^2=np(1-p)$ | 동전 20번 던져 앞면 나오는 횟수 |
| **기하 (Geometric)** | 처음 성공할 때까지의 시행 횟수 | 처음 앞면 나올 때까지 던진 횟수 |
| **음이항 (Negative Binomial)** | $r$번 성공할 때까지의 시행 횟수 | 앞면 5번 나올 때까지 던진 횟수 |
| **초기하 (Hypergeometric)** | **비복원추출**로 $n$번 뽑았을 때 원하는 결과 $k$번 | 10개 중 당첨 4개일 때 4번 뽑아 당첨 2개 |
| **다항 (Multinomial)** | $n$번 시행에서 각 시행이 여러 결과를 가질 수 있음 | 주사위 20번 던져 각 면 나오는 횟수 |
| **포아송 (Poisson)** | 단위 시간 내 사건 발생 횟수 | 하루 출생자 수, 1시간 전화 건수 |

> [!TIP] 암기법: **누포항하** (베르**누**이, **포**아송, 이**항**, 기**하**)

#### ● 연속 확률분포 (Continuous Probability Distribution)

값을 **셀 수 없는** 분포이며 **확률밀도함수(PDF)**로 표현합니다.

| 분포 | 정의 | 활용 |
| :--- | :--- | :--- |
| **균일 (Uniform)** | 모든 값이 동일한 확률 | - |
| **정규 (Normal)** | 일상에서 흔한 확률변수의 평균 분포 근사. $N(\mu, \sigma^2)$ | 키, IQ, 시험 성적 분포 |
| **표준정규 (Standard Normal)** | 평균 0, 표준편차 1. $Z = \frac{X-\mu}{\sigma} \sim N(0,1)$ | 표준화, z검정 |
| **t-분포** | 정규분포와 유사하나 **꼬리가 더 두껍고 긴** 분포 | **t검정:** 표본 30개 미만일 때 평균 검정 |
| **카이제곱 ($\chi^2$)** | 독립적인 정규분포 변수들의 **제곱합** | **$\chi^2$ 검정:** 두 집단 동질성, 단일 집단 모분산 검정 |
| **F-분포** | 두 개의 **카이제곱분포의 비율** | **F검정:** 두 집단 분산 동질성 / **ANOVA:** 세 집단 이상 검정 |

### 3. 표본분포 (Sampling Distribution)

#### ● 중심극한정리 (Central Limit Theorem, CLT) ⭐ 매우 중요

> [!WARNING] 🔴 핵심 개념
> **모집단의 분포에 상관없이**, 표본크기가 **충분히 크면(30개 이상)** 표본평균의 분포는 **정규분포**에 근사합니다.
> 
> 즉, **어떤 모집단**에서 추출된 표본이든, 표본평균의 분포는 정규분포를 이룹니다. 이는 추론통계의 기반이 됩니다.

#### ● 표본평균의 표본분포

($\mu$: 모집단 평균, $\sigma$: 모집단 표준편차, $\bar{X}$: 표본평균, $n$: 표본 크기)

* **(1) 평균:** $E(\bar{X}) = \mu$
* **(2) 분산:** $V(\bar{X}) = \frac{\sigma^2}{n}$
* **(3) 표준오차:** $SE(\bar{X}) = \frac{\sigma}{\sqrt{n}}$
* **(4) 표준화:** $Z = \frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \sim N(0,1)$

#### ● 표본비율의 표본분포

($\hat{P}$: 표본비율, $P$: 모집단 비율, $q = 1-P$)

* **(1) 표본비율:** $\hat{P} = \frac{X}{n}$ ($X$: 성공 횟수)
* **(2) 평균:** $E(\hat{P}) = P$
* **(3) 표준오차:** $SE(\hat{P}) = \sqrt{\frac{pq}{n}}$
* **(4) 표준화:** $Z = \frac{\hat{P} - P}{\sqrt{pq/n}} \sim N(0,1)$

### 4. 추론통계 (Inferential Statistics)

모집단 → 표본추출 → 표본집단. 표본통계량으로 **모수(모집단의 특성)**를 추정합니다.

#### ● 모수와 추정량 (Parameter & Estimator)

| 구분 | 설명 | 예시 |
| :--- | :--- | :--- |
| **모수 (Parameter)** | **모집단**의 특성을 나타내는 **고정된 값**. 직접 관측 불가. | 모평균 $\mu$, 모분산 $\sigma^2$, 모비율 $p$ |
| **추정량 (Estimator)** | **표본**으로부터 모수를 추정하기 위해 사용하는 **통계량(함수)**. 표본마다 값이 달라짐. | 표본평균 $\bar{X}$, 표본분산 $s^2$, 표본비율 $\hat{p}$ |
| **추정값 (Estimate)** | 추정량에 **실제 표본 데이터**를 대입한 결과값. | $\bar{x} = 72.5$ (구체적 숫자) |

> [!INFO] 💡 구분 요약
> **모수** = 알 수 없는 진짜 값 (모집단) / **추정량** = 그 값을 추측하는 공식 (표본 기반) / **추정값** = 공식에 넣어 나온 결과

#### ● 점추정 (Point Estimation)

모수를 **특정한 하나의 값**으로 추정. 추정량(Estimator)으로 모수를 추정합니다.

* **추정량의 조건 (불효일충):**
    * **불편성 (Unbiasedness):** $E(\hat{\theta}) = \theta$ (편향 0)
    * **효율성 (Efficiency):** 분산이 작을수록 효율적
    * **일치성 (Consistency):** 표본 크기 ↑ → 추정량이 모수에 수렴
    * **충족성 (Sufficiency):** 모집단 정보를 최대한 반영

> [!TIP] 암기법: **불효일충**

* **대표적인 추정량:**
    * 모평균 $\mu$ → 표본평균 $\bar{X} = \frac{1}{n}\sum X_i$
    * 모분산 $\sigma^2$ → 표본분산 $s^2 = \frac{1}{n-1}\sum(X_i - \bar{X})^2$
    * 모비율 $p$ → 표본비율 $\hat{p} = X/n$

> [!INFO] 💡 표본분산은 왜 $n-1$로 나누나? (베셀 보정, Bessel's Correction)
> * **불편성 확보:** $\bar{X}$(표본평균)를 쓰면 편차의 제곱합이 **진짜 평균 $\mu$**를 쓸 때보다 **과소평가**됩니다. $n$으로 나누면 $E(s^2) < \sigma^2$가 되어 편향이 생깁니다. $n-1$로 나누면 $E(s^2) = \sigma^2$가 되어 **불편추정량**이 됩니다.
> * **자유도 (df):** 평균 $\bar{X}$를 이미 추정했기 때문에, $n$개 중 $n-1$개만 독립적으로 변할 수 있습니다. (나머지 하나는 평균 조건으로 고정됨)

* **평균제곱오차 (MSE):** $MSE(\hat{\theta}) = E[(\hat{\theta} - \theta)^2] = Var(\hat{\theta}) + [Bias(\hat{\theta})]^2$

#### ● 구간추정 (Interval Estimation)

모수를 **특정 구간**으로 추정. 95%, 99% 신뢰수준을 가장 많이 사용합니다.

* **모분산을 아는 경우 (Z 사용):**
    * $\bar{X} - Z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}} \leq \mu \leq \bar{X} + Z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}$
    * 95%: $Z_{\alpha/2} = 1.960$ / 99%: $Z_{\alpha/2} = 2.576$
* **모분산을 모르는 경우 (t 사용):**
    * 자유도 $n-1$인 **t분포** 이용.
    * $\bar{X} - t_{\alpha/2, n-1} \cdot \frac{S}{\sqrt{n}} \leq \mu \leq \bar{X} + t_{\alpha/2, n-1} \cdot \frac{S}{\sqrt{n}}$

#### ● 가설검정 (Hypothesis Testing)

모집단의 특성에 대한 **주장을 가설로 세우고**, 표본조사로 **채택·기각 여부**를 판정합니다.

* **(1) 귀무가설 ($H_0$):** 일반적으로 가정하는 가설. **차이가 없다**는 전제.
* **(2) 대립가설 ($H_1$):** 귀무가설을 기각할 때 채택. **증명하고자 하는** 가설 (차이가 있다, 크다/작다).
* **(3) 검정통계량:** 귀무가설 기각 여부를 결정하는 표본 통계량. **p-value**로 판단.
* **(4) 유의수준 ($\alpha$):** 귀무가설이 참일 때 기각하는 **1종 오류** 확률의 허용 한계. (일반적 0.05)
* **(5) 기각역:** 귀무가설이 기각되고 대립가설이 채택되는 검정통계량의 영역.

#### ● 검정결과와 오류 (1종·2종 오류)

|  | $H_0$ 사실이라고 판정 | $H_0$ 거짓이라고 판정 |
| :--- | :--- | :--- |
| **$H_0$가 사실** | 옳은 결정 | **1종 오류 ($\alpha$)** — Type I Error |
| **$H_0$가 거짓** | **2종 오류 ($\beta$)** — Type II Error | 옳은 결정 |

> [!TIP] 암기법
> **1종 오류:** 참인데 기각함 (오탐, false positive) / **2종 오류:** 거짓인데 채택함 (미탐, false negative)

#### ● 가설 검정의 절차

* **1. 가설 설정**
    * **귀무가설 ($H_0$):** 차이가 없다, 효과가 없다, 독립이다.
    * **대립가설 ($H_1$):** 차이가 있다, 효과가 있다, 독립이 아니다.
* **2. 검증 방향 결정**
    * 대립가설이 **"같지 않다"** → **양측검정** (Two-tailed)
    * 대립가설이 **"크다"** 또는 **"작다"** → **단측검정** (One-tailed)
* **3. 검정 방법 및 표본 유형**
    * 1개 모집단 → **일표본 검정**
    * 2개 모집단 → **이표본 검정**
    * 3개 이상 집단 → **ANOVA 검정**
* **4. 검정통계량 계산과 기각역 판단**
    * 검정통계량이 **기각역**에 있으면 → **$H_0$ 기각**
    * 검정통계량이 **채택역**에 있으면 → **$H_0$ 채택**

#### ● t검정 유형 (단일·대응·독립표본)

| 유형 | 적용 상황 |
| :--- | :--- |
| **단일표본 (One-sample)** | 모집단에 대한 평균 검정 |
| **대응표본 (Paired-sample)** | **동일** 모집단에 대한 평균 비교 (전·후, 쌍 비교) |
| **독립표본 (Independent-sample)** | **서로 다른** 모집단에 대한 평균 비교 |

#### ● 가설 검정 문제 풀이 예시

**문제:** A공장 불량률 1.7%, 모표준편차 0.3%로 관리 중. 9개 샘플 분석 시 평균 1.5%. "불량률 1.7%" 주장에 5% 유의수준으로 가설검정. ($Z_{0.05}=1.645$, $Z_{0.025}=1.96$)

* **1. 가설 설정:** $H_0: \mu = 1.7$% (차이가 없다)
* **2. 검증 방향:** 대립가설 "같지 않다" → **양측검정**
* **3. 검정 방법:** 1개 모집단 → **일표본 Z검정** (모분산 알 때)
* **4. 검정통계량·판정:**
    * $Z_0 = \frac{\bar{X} - \mu}{\sigma/\sqrt{n}} = \frac{1.5 - 1.7}{0.3/\sqrt{9}} = \frac{-0.2}{0.1} = -2.0$
    * 양측 α=0.05 → 기각역 $|Z| > 1.96$
    * $Z_0 = -2.0 < -1.96$ → **귀무가설 기각**
* **결론:** 5% 유의수준에서 불량률 1.7%라는 주장을 기각. 표본은 1.7%와 통계적으로 유의한 차이를 보임.

---

**(심화) 모표준편차를 모르는 경우 — t검정**

동일 문제에서 **모표준편차(σ) 대신 표본표준편차(S=0.3%)**를 사용할 때 → **t검정** 적용.

* **1)~3) 동일** (가설, 양측검정, 일표본)
* **4) 검정통계량·판정:**
    * $t_0 = \frac{\bar{X} - \mu}{S/\sqrt{n}} = \frac{1.5 - 1.7}{0.3/\sqrt{9}} = -2.0$
    * 자유도 $df = n-1 = 8$. 양측 α=0.05 → 기각역 $\pm t_{0.025}(8) = \pm 2.306$
    * $t_0 = -2.0 > -2.306$ → **귀무가설 채택** (기각역 밖)
* **5) 표본 유형:** 모집단 평균 검정 → **단일표본**

> [!NOTE] Z검정 vs t검정
> 같은 데이터(1.5 vs 1.7)여도 **Z검정**에서는 기각, **t검정**에서는 채택. t분포는 꼬리가 더 두꺼워 기각역이 넓어지므로(±2.306 > ±1.96), 모분산을 추정할 때의 불확실성이 반영됩니다.
