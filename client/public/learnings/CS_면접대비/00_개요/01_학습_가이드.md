# CS 면접대비 — 학습 가이드

날짜: 2026-07-03  
태그: [면접, CS, 운영체제, 네트워크, 자료구조, 데이터베이스, 파이썬, 타입스크립트]

---

## 목적

- 모빌리티·차량용 SW·웹 백엔드 면접에서 자주 나오는 CS 기초를 **질문–답변 형식**으로 정리
- 각 답변은 **30초~1분** 분량의 구두 답변을 목표로 작성 (핵심 키워드 + 꼬리질문 대비 포함)
- 본인 경험을 붙일 수 있는 문항은 `[경험 연결]` 표시

## 문서 형식

각 문항은 아래 구조를 따릅니다.

```markdown
### Qn. 질문 제목

**질문** — 면접관이 던질 수 있는 원문

**답변** — 구두 답변 예시 (1~3문단)

**핵심 키워드** — 한 줄 암기용

**꼬리질문** — 심화·후속 질문과 짧은 답
```

## 학습 팁

1. **키워드 먼저** — 답변 전에 핵심 키워드 3개를 머릿속에 정리한 뒤 말하기
2. **비교 질문** — A vs B 형태는 표로 암기 (장단점, 사용 상황)
3. **경험 연결** — `[경험 연결]` 문항은 본인 프로젝트(Express API, Supabase, 차량 데이터 등)에 맞게 1~2문장 추가
4. **모빌리티 맥락** — 차량·실시간·대용량 로그 관점에서 예시를 한 번 더 말하면 차별화됨
