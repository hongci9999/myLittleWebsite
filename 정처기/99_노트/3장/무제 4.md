## sql

관계대수를 기초로 만들어짐
자연어와 유사한 구조

내장 sql
호스트 프로그램이 실행 될 때 함께ㅐ 실행
일반 dql은 다수의 튜플 반환 가능하지만 내장 sql은 하나의 튜플만 반환
호스트 변수와 대응되는 데이터베이스 필드 이름은 달라도 되지만 데이터 타입은 일치해야함


sql 명령어 분류

한 번 수행되면 이전 상태로 되돌릴 수 없음

create - 객체 생성
alter - 객체 수정
drop - 객체 제거
truncate - 객체 데이터 전체 삭제

데이터 조작어 dml

rollback - 트랜잭션 작업 취소
checkpoint - 트랜잭션 복귀지점 설정
