# 분석모형 평가 및 개선 (Evaluation and Improvement of Analysis Models)

## 1. 분석모형 평가 (Analysis Model Evaluation)

### 분류모델 평가지표 (Classification Model Evaluation Metrics)

---

### (1) 혼동행렬 (Confusion Matrix / 오분류표)

**예측값(Predicted)**과 **실제값(Actual)**의 관계를 나타내는 2×2 행렬.

|  | Actual: TRUE | Actual: FALSE |
| :--- | :--- | :--- |
| **Predicted: TRUE** | TRUE POSITIVE (TP) | FALSE POSITIVE (FP) |
| **Predicted: FALSE** | FALSE NEGATIVE (FN) | TRUE NEGATIVE (TN) |

#### 용어 해석
| 용어 | 의미 |
| :--- | :--- |
| **TRUE / FALSE** | 예측이 실제값과 **일치 여부** (맞음=TRUE, 틀림=FALSE) |
| **POSITIVE / NEGATIVE** | **예측한 클래스** (양성 예측=POSITIVE, 음성 예측=NEGATIVE) |

> [!TIP] 한 줄 요약
> **TRUE/FALSE** = 맞춤/틀림, **POSITIVE/NEGATIVE** = 예측한 답이 양성/음성

---

### (2) 평가지표 (Evaluation Metrics)

| 지표 | 설명 | 계산식 |
| :--- | :--- | :--- |
| **정밀도 (Precision)** | 양성으로 예측한 것 중 실제 양성 비율 | $\frac{TP}{TP + FP}$ |
| **재현율 (Recall)** | 실제 양성 중 맞게 예측한 비율 | $\frac{TP}{TP + FN}$ |
| **특이도 (Specificity)** | 실제 음성 중 맞게 예측한 비율 | $\frac{TN}{FP + TN}$ |
| **정확도 (Accuracy)** | 전체 중 올바르게 예측한 비율 | $\frac{TP + TN}{TP + FP + FN + TN}$ |
| **FP Rate (False Alarm Rate)** | 음성인데 양성으로 잘못 예측한 비율 | $\frac{FP}{FP + TN}$ |
| **F-1 Score** | 정밀도와 재현율의 조화평균 | $2 \times \frac{Precision \times Recall}{Precision + Recall}$ |
| **F-β Score** | β에 따라 재현율·정밀도 비중 조절 | $(1 + \beta^2) \times \frac{Precision \times Recall}{(\beta^2 \cdot Precision) + Recall}$ |

> [!NOTE] 재현율의 다른 이름
> Recall = **Sensitivity** = **TP Rate** = **Hit Rate** (= TPR)

> [!IMPORTANT] F-1 Score
> **정밀도(Precision)와 재현율(Recall)의 조화평균**이다.

> [!TIP] 정밀도·재현율 트레이드오프
> 정밀도와 재현율은 **상충(trade-off) 관계**이다. 한쪽을 높이면 다른 한쪽이 희생된다.

#### F-β Score 해석
* **β > 1**: 재현율(Recall)을 더 중요하게
* **β < 1**: 정밀도(Precision)를 더 중요하게
* **β = 1**: F-1 Score와 동일

---

### (3) ROC Curve (ROC 커브)

| 항목 | 설명 |
| :--- | :--- |
| **X축** | 1 - Specificity = **FPR** (False Positive Rate) |
| **Y축** | Sensitivity = **TPR** (True Positive Rate) |
| **AUC** | 곡선 아래 면적. 0.5 ~ 1 범위, **1에 가까울수록** 모델 성능이 좋음 |

---

### (4) Lift Chart (이익도표)

| 항목 | 설명 |
| :--- | :--- |
| **내용** | 등급/십분위(decile)별로 나눠 **응답탐지율**, **응답률**, **향상도(lift)** 등을 표시 |
| **Lift Curve (향상도 곡선)** | Lift Chart를 시각화한 곡선 |

---

## 2. 회귀모델 평가지표 (Regression Model Evaluation Metrics)

### (1) 손실함수 / 비용함수 (Loss Functions / Cost Functions)

> [!NOTE] 핵심 개념
> **정답과의 오차**를 측정하는 함수. 값이 작을수록 모델이 좋다.

| 지표 | 계산식 |
| :--- | :--- |
| **MSE** (Mean Squared Error) | $MSE = \frac{1}{n} \sum(y - \hat{y})^2$ |
| **MAE** (Mean Absolute Error) | $MAE = \frac{1}{n} \sum \|y - \hat{y}\|$ |
| **RMSE** (Root Mean Squared Error) | $RMSE = \sqrt{MSE}$ |
| **MAPE** (Mean Absolute Percentage Error) | $MAPE = \frac{1}{n} \sum \|\frac{y - \hat{y}}{y}\| \times 100$ |
| **MPE** (Mean Percentage Error) | $MPE = \frac{1}{n} \sum (\frac{y - \hat{y}}{y}) \times 100$ |

* $n$: 샘플 수, $y$: 실제값, $\hat{y}$: 예측값

---

### (2) 결정계수 (Coefficient of Determination)

| 지표 | 계산식 |
| :--- | :--- |
| **R² (R-Square)** | $R^2 = \frac{SSR}{SST} = 1 - \frac{SSE}{SST}$ |
| **Adjusted R²** (수정된 결정계수) | $Adjusted\ R^2 = 1 - (n - 1) \frac{MSE}{SST}$ |

#### 제곱합 용어
| 용어 | 의미 | 계산식 |
| :--- | :--- | :--- |
| **SSR** | 회귀 제곱합 | $\sum(\hat{y} - \bar{y})^2$ |
| **SSE** | 오차 제곱합 (잔차) | $\sum(y - \hat{y})^2$ |
| **SST** | 총 제곱합 | $SSR + SSE$ 또는 $\sum(y - \bar{y})^2$ |

* $\bar{y}$: 실제값의 평균. R²는 1에 가까울수록 설명력이 높다.

---

## 3. 군집분석의 평가지표 (Cluster Analysis Evaluation Metrics)

> [!TIP] 좋은 군집모델의 기준
> **같은 군집 내**에는 응집도가 **높고**, **서로 다른 군집 간**에는 응집도가 **낮아야** 좋다.

---

### (1) Dunn Index (던 인덱스)

| 항목 | 설명 |
| :--- | :--- |
| **정의** | 군집 간 거리의 **최솟값** ÷ 군집 내 거리의 **최댓값** |
| **해석** | **값이 클수록** 좋음. 군집들이 촘촘하고 서로 멀리 떨어져 있음 |

---

### (2) 실루엣 계수 (Silhouette Coefficient)

$$S(i) = \frac{b(i) - a(i)}{\max(a(i), b(i))}$$

| 기호 | 의미 |
| :--- | :--- |
| $a(i)$ | $i$번째 데이터와 **같은 군집** 내 다른 점들 간의 **평균 거리** (군집 내 거리) |
| $b(i)$ | $i$번째 데이터와 **가장 가까운 다른 군집** 내 점들 간의 **평균 거리** (군집 간 거리) |

| 값 범위 | 해석 |
| :--- | :--- |
| **1에 가까움** | 해당 점이 군집에 잘 속함 |
| **0 근처** | 군집 경계에 있음 |
| **음수** | 잘못된 군집에 배정되었을 가능성 |

---

### (3) WCSS (Within Clusters Sum of Squares)

$$WCSS = \sum_{k=1}^{K} \sum_{d_i \in C_k} \text{distance}(d_i, C_k)^2$$

| 기호 | 의미 |
| :--- | :--- |
| $C_k$ | $k$번째 군집의 **중심(centroid)** |
| $d_i$ | 군집 내 **데이터 포인트** |

* 각 데이터와 **할당된 군집 중심** 사이의 **제곱 거리**의 합. **값이 작을수록** 군집이 촘촘함.

---

### (4) Elbow Method (팔꿈치 방법)

**최적의 군집 개수(K)** 선택 방법.

| 항목 | 설명 |
| :--- | :--- |
| **X축** | 군집의 수 K (1, 2, 3, …) |
| **Y축** | WCSS |
| **최적 K** | WCSS 그래프에서 **경사가 완만해지는 지점**(팔꿈치 모양) → 그 K가 최적 |

---

## 4. 분석모형 진단 (Model Diagnosis)

### 과대적합 vs 과소적합

| 유형 | 편향·분산 | 특징 |
| :--- | :--- | :--- |
| **과대적합 (Overfitting)** | **높은 분산**, 낮은 편향 | 훈련 데이터에 과도하게 맞춤. 일반화 성능 저하 |
| **과소적합 (Underfitting)** | 낮은 분산, **높은 편향** | 모델이 너무 단순. 훈련·검증 모두 성능 부족 |

> [!NOTE] 최적 모델 선정
> **분산과 편향의 trade-off**를 확인하여 최적의 모델을 선정한다.

---

## 5. 교차 검증 (Cross-Validation)

| 방법 | 설명 |
| :--- | :--- |
| **(1) Hold-out (홀드아웃)** | 데이터를 한 번만 훈련·검증 세트로 분할 |
| **(2) K-fold 교차검증** | K개 그룹으로 나누고, **K-1개로 학습 · 1개로 검증**을 K번 반복 |
| **(3) LOOCV** (Leave-One-Out) | **1개 데이터만** 검증에 사용, 나머지 전체로 학습. 신뢰도 높지만 **계산 비용 큼** |
| **(4) Bootstrapping (부트스트래핑)** | 복원추출로 샘플 생성. 데이터 부족·불균형 해결에 활용 |

---

## 6. 적합도 검정 (Goodness of Fit Test)

| 검정 | 대상/방법 | 설명 |
| :--- | :--- | :--- |
| **(1) Q-Q Plot** | 정규성 시각 검정 | **분포의 분위수**를 활용. 정규성이 있으면 **45° 대각선** 근처에 점이 분포 |
| **(2) 카이제곱 검정** | 범주형 | 변수를 **범주(category)**로 묶어 적합도 검정 |
| **(3) Shapiro-Wilk 검정** | 정규성 | 데이터와 **표준정규 점수** 간 **선형 상관**을 측정 |
| **(4) Kolmogorov-Smirnov 검정** | 연속형 | **누적 분포함수(CDF)**를 비교하여 검정 |
