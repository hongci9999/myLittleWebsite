# 고급 분석기법 (Advanced Analysis Techniques)

## 📌 핵심 요약
**범주형 자료 분석**은 **분할표(교차표)**를 활용하여 범주형 변수 간 관계를 파악합니다. **상대 위험도(RR)**와 **오즈비(OR)**로 위험도를 비교하고, **카이제곱 검정**으로 적합도·독립성·동질성을 검정합니다. **다변량 분석**은 여러 독립변수 → 여러 종속변수 관계를 다룹니다. **ANOVA**는 집단 평균 비교, **분산분석**은 일원·이원으로 구분, **MANOVA**는 다수의 종속변수와 집단 분석입니다. **요인분석**은 변수를 **축약**하여 소수의 요인으로 줄이고, **정준상관분석**은 두 변수군 간 **선형결합**으로 상관을 최대화합니다. **시계열 분석**은 시간의 흐름에 따라 관찰된 자료(주가·기온 등)의 특성을 파악하여 **미래를 예측**합니다. 예측을 위해 **정상성**(일정한 평균·분산)이 필요하며, **차분**·**변환**으로 정상화하고, **평활화**(이동평균·지수평활)로 패턴을 파악합니다. **베이즈 정리**는 사전확률과 우도로 **사후확률**을 구하며, **나이브베이즈**는 특성의 **독립성 가정** 하에 베이즈 정리를 활용한 분류 기법입니다.

---

## 📝 상세 정리

### 1. 범주형 자료 분석 (Categorical Data Analysis)

#### ● 분할표 (교차표, Contingency Table)

여러 개의 **범주형 변수**를 기준으로 관측치를 기록하는 표.

#### ● 상대 위험도 (Relative Risk, RR)

$$RR = \frac{\pi_1}{\pi_2}$$

* $\pi_1$: 노출군(또는 실험군)에서 사건 발생 확률
* $\pi_2$: 비노출군(또는 대조군)에서 사건 발생 확률

#### ● 오즈비 (Odds Ratio, OR)

$$OR = \frac{Odds_{p_1}}{Odds_{p_2}} = \frac{P_1/(1-P_1)}{P_2/(1-P_2)}$$

---

### 2. 계산 예시: 알콜중독 vs 심장질환

**분할표:**

| | 심장질환 O | 심장질환 X | 합계 |
| :--- | :---: | :---: | :---: |
| **알콜중독 O** | 14 (0.7) | 6 (0.3) | 20 |
| **알콜중독 X** | 10 (0.2) | 40 (0.8) | 50 |

* **상대 위험도(RR):** $RR = \frac{\pi_1}{\pi_2} = \frac{0.7}{0.2} = 3.5$
* **오즈비(OR):** $OR = \frac{0.7/0.3}{0.2/0.8} = \frac{2.33}{0.25} = 9.33$

> [!NOTE] 해석
> RR=3.5 → 알콜중독인 경우 심장질환 위험이 3.5배. OR=9.33 → 오즈비가 1보다 크면 양의 연관.

---

### 3. 적합도·독립성·동질성 검정

**카이제곱 검정**을 활용하여 **관측빈도**와 **기대빈도**의 차이를 검정.

| 검정 | 목적 |
| :--- | :--- |
| **적합도 검정** | 관측된 데이터가 **특정 분포**에 적합한지 검정 |
| **독립성 검정** | 두 변수 간 **독립성** 여부 검정 |
| **동질성 검정** | 두 개 이상의 집단이 **동일한 분포**를 따르는지 검정 |

> [!TIP] 교차표 활용
> **독립성 검정**과 **동질성 검정**은 **교차표(분할표)**를 활용함.

---

### 4. 다변량 분석 (Multivariate Analysis)

여러 **독립변수** $[X_1, X_2, \dots, X_n]$와 여러 **종속변수** $[Y_1, Y_2, Y_3, \dots]$ 간의 관계를 분석하는 기법.

---

### 5. 분산분석 (ANOVA/MANOVA)

| 구분 | 설명 |
| :--- | :--- |
| **(1) ANOVA (분산분석)** | 여러 집단의 **평균**을 비교하여 차이가 통계적으로 유의한지 검정 |
| **일원분산분석** | **한 요인**과 여러 집단 간 분석 |
| **이원분산분석** | **두 요인**과 여러 집단 간 분석 |
| **(2) MANOVA (다변량분산분석)** | **다수의 종속변수(요인)**와 여러 집단 간 분석 |

---

### 6. 요인분석 (Factor Analysis)

**정의:** 변수 간 **상관관계**를 분석하여, 한 그룹 내 변수들을 **축약**하여 소수의 **요인**으로 줄이는 기법.

#### ● 요인추출방법

| 방법 | 설명 |
| :--- | :--- |
| **PCA (주성분분석)** | **총분산**을 기준으로 요인 추출. **가장 많이 사용** |
| **주축요인분석** | **공통분산**만을 기준으로 요인 추출 |
| **최소제곱요인분석** | **잔차를 최소화**하여 요인 추출 |

#### ● 요인회전 (Factor Rotation)

| 유형 | 방법 예시 |
| :--- | :--- |
| **직각회전 (Orthogonal)** | VARIMAX, QUARTIMAX, EQUIMAX |
| **사각회전 (Oblique)** | OBLIMIN, PROMAX |

---

### 7. 정준상관분석 (Canonical Correlation Analysis)

**정의:** 두 개 이상의 **변수군** 간 상관관계를 가장 잘 설명하는 **선형결합**을 찾아 분석하는 기법.

* **목적:** 두 변수군 간 관계를 **선형결합**으로 최대화하는 최적의 조합 도출

---

### 8. 시계열 분석 (Time Series Analysis)

**정의:** **시간의 흐름에 따라 관찰된 자료**(주가데이터, 기온데이터 등)의 특성을 파악하여 **미래를 예측**하는 기법.

---

#### ● 정상성 (Stationarity)

시계열 예측을 위해 **정상성**이 충족되어야 함 — 모든 시점에서 **일정한 평균과 분산**을 가져야 함.

**정상 시계열로 변환하는 방법:**

| 방법 | 설명 |
| :--- | :--- |
| **(1) 차분 (Differencing)** | 현재값에서 **이전값을 빼서** 평균을 일정하게 만드는 방법 |
| **(2) 변환 (Transformation)** | **Log**, **제곱근**, **Box-Cox** 변환 등으로 분산을 안정화 |

---

#### ● 평활화 (Smoothing)

**슬라이딩 윈도우**를 이용해 시계열의 패턴을 파악하는 방법.

| 방법 | 설명 |
| :--- | :--- |
| **이동평균법 (Moving Average)** | 일정 기간의 **평균**을 사용 |
| **지수평활법 (Exponential Smoothing)** | **최근 데이터에 더 큰 가중치** 부여 |

---

#### ● 백색 잡음 (White Noise)

시계열 모형의 **오차항**. 다음을 만족해야 함:

* **평균·분산이 일정**
* **자기상관 없음**
* **정상성 만족**

**가우시안 백색 잡음 (Gaussian White Noise):** 평균이 0인 백색 잡음

---

#### ● 시계열 모형 (Time Series Models)

| 모형 | 설명 | 선택 기준 |
| :--- | :--- | :--- |
| **AR (자기회귀)** | **과거 시계열 값**이 미래 값을 결정 | **PACF** 사용. PACF가 $p+1$에서 급격히 감소 → AR(p) 선택 |
| **MA (이동평균)** | 과거 **백색 잡음 오차항**의 선형결합 | **ACF** 사용. ACF가 $q+1$에서 급격히 감소 → MA(q) 선택 |
| **ARIMA** | AR + MA 조합. $ARIMA(p, d, q)$ | $p$: AR 차수, $q$: MA 차수, $d$: **차분 횟수** (정상성 확보) |

**ARIMA 특수 케이스:**

| 조건 | 모형 |
| :--- | :--- |
| $d=0$ | **ARMA** |
| $p=0$ | **IMA** |
| $q=0$ | **ARI** |

> [!NOTE] 계절성
> **계절성**이 있으면 **SARIMA** 모형 고려.

#### ● 시계열 모형 예시

**AR(1):** $Y_t = \phi_1 Y_{t-1} + \varepsilon_t$ — 직전 시점 값이 현재값에 영향  
예: $Y_t = 0.7 Y_{t-1} + \varepsilon_t$ → 어제 주가가 오늘 주가 예측에 70% 반영

**MA(1):** $Y_t = \varepsilon_t + \theta_1 \varepsilon_{t-1}$ — 직전 시점 오차가 현재값에 영향  
예: $Y_t = \varepsilon_t + 0.5 \varepsilon_{t-1}$ → 어제의 예측 오차가 오늘에 반영

**ARIMA(1,1,1):** 1차 차분 후 AR(1) + MA(1) 적용  
예: 추세가 있는 주가 → 차분으로 추세 제거 → 정상 시계열로 ARMA 모형 적합

| 모형 | 수식 예시 | 용도 |
| :--- | :--- | :--- |
| **AR(2)** | $Y_t = \phi_1 Y_{t-1} + \phi_2 Y_{t-2} + \varepsilon_t$ | 최근 2시점이 현재에 영향 |
| **MA(2)** | $Y_t = \varepsilon_t + \theta_1 \varepsilon_{t-1} + \theta_2 \varepsilon_{t-2}$ | 최근 2시점 오차가 반영 |
| **ARMA(1,1)** | $Y_t = \phi_1 Y_{t-1} + \varepsilon_t + \theta_1 \varepsilon_{t-1}$ | AR + MA 혼합 |

---

#### ● 분해시계열 (Decomposition of Time Series)

시계열을 구성 요인으로 분리하여 분석하는 방법.

| 요인 | 설명 |
| :--- | :--- |
| **(1) 추세 요인 (Trend)** | 장기적 **상승·하락** 경향 |
| **(2) 계절 요인 (Seasonal)** | **고정된 주기**의 변동 (계절 등) |
| **(3) 순환 요인 (Cyclical)** | 주기가 **불규칙하거나 불명확**한 변동 (경기순환, 산업 트렌드 등) |
| **(4) 불규칙 요인 (Irregular)** | **무작위** 또는 위 세 요인으로 설명되지 않는 요인 |

> [!TIP] 암기법
> **추운 계절의 순환이 불규칙하다** — 추(세), 계(절), 순(환), 불(규칙)

### 9. 베이지안 기법 (Bayesian Techniques)

#### ● 베이즈 정리 (Bayes' Theorem)

$$P(A|B) = \frac{P(B|A)P(A)}{P(B)} = \frac{P(A \cap B)}{P(B)}$$

| 기호 | 의미 |
| :--- | :--- |
| $P(A|B)$ | **사후확률** (Posterior) |
| $P(B|A)$ | **우도** (Likelihood) |
| $P(A)$ | **사전확률** (Prior) |
| $P(B)$ | **주변우도** (Marginal likelihood / Evidence) |

---

#### ● 베이즈 정리 계산 예시: 대학 합격

**조건:** 남학생 60%(합격률 30%), 여학생 40%(합격률 50%). **질문:** 합격한 신입생이 남학생일 확률 $P(\text{남학생}|\text{합격})$?

**우도표 (결합확률):**

| | 합격 | 불합격 |
| :--- | :---: | :---: |
| **남학생** | $0.6 \times 0.3 = 0.18$ | $0.6 \times 0.7 = 0.42$ |
| **여학생** | $0.4 \times 0.5 = 0.20$ | $0.4 \times 0.5 = 0.20$ |

* $P(\text{합격}) = P(B) = 0.18 + 0.20 = 0.38$

$$P(\text{남학생}|\text{합격}) = \frac{P(A \cap B)}{P(B)} = \frac{0.18}{0.38} \approx 0.47$$

---

#### ● 나이브베이즈 분류 (Naive Bayes Classification)

**정의:** **나이브(독립)** + **베이즈 정리**. 베이즈 정리를 이용해 항목이 특정 범주에 속할 확률을 계산하여 분류.

> [!IMPORTANT] 핵심 가정
> **특성(변수)들이 서로 독립적**이라는 가정이 필요함.
