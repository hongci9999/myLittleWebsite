# 클래스 설계
## 클래스
자바는 모든 코드를 클래스 단위로 구현

멤버변수 - 객체의 상태, 수치,튿성을 나타내는 변수
멤버 메소드 - 객체에서 발생하는 모든 제어 및 기능, 요청행위

기본 출력 메소ㅓ드
print() 등

접근 제한자
접근 수준을 결정하는 토느
public- 프록램 내 모든 객체에서 접근
default - 같은 패키지 내에서만 접근
protected - 하위 클래스에서 접근
private - 자신 클래스만 접근

멤버 접근

객체변수. 멤버이름

멤버변수와 지역변수

이름이 같으면 this(이 클래스의) 키워드로 구분
