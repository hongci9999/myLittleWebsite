# 학습자료 구성 가이드

날짜: 2026-04-14
중요도: 상

## 1) 핵심 개념

- 본 문서는 정보처리기사 실기 범위를 과목 기준으로 분해한 정리 자료다.
- 각 파일은 "핵심 개념 -> 필수 키워드 -> 비교 정리 -> 예시 -> 한 페이지 요약" 순서로 통일한다.

## 2) 필수 키워드

- 체계화 = 과목/주제/키워드 단위로 분리해 탐색 시간을 줄이는 방식
- 단답형 = 한 줄 정의 또는 비교표로 빠르게 확인 가능한 유형
- 계산형 = 공식과 단계 절차를 함께 외워야 하는 유형

## 3) 비교 정리

| 구분 | 기존 분산 노트 | 체계형 정리 |
|---|---|---|
| 탐색성 | 파일 위치를 기억해야 함 | 폴더명만으로 범위 파악 가능 |
| 일관성 | 문서마다 형식이 다름 | 공통 섹션 템플릿 사용 |
| 확장성 | 추가 시 중복 가능 | 주제 단위로 파일 추가 가능 |

## 4) 예시

- 데이터베이스를 볼 때는 `02_데이터베이스`를 먼저 보고, SQL 세부 문법은 `06_SQL_프로그래밍_재정리`를 이어서 확인한다.

## 5) 한 페이지 요약

- 과목별 분류 + 공통 템플릿 + SQL 재정리 분리 원칙을 유지한다.
- 기존 SQL 폴더와 중복되어도 학습 효율을 위해 별도 정리본을 유지한다.
