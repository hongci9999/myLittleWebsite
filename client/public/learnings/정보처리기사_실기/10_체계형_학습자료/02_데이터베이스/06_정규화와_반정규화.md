# 정규화와 반정규화

## 핵심 요약

정규화는 데이터 중복을 줄이고 이상 현상을 제거하기 위한 분해 과정이다. 실기에서는 각 정규형의 이름만 외우는 것이 아니라, 어떤 종속성을 제거하는 단계인지 이해해야 한다.

## 왜 중요한가

- 정규화는 거의 매년 반복되는 핵심 주제다.
- 2NF와 3NF, 3NF와 BCNF를 헷갈리는 경우가 많다.
- 함수 종속과 이상 현상을 연결해서 이해해야 실전 문제를 풀 수 있다.

## 1. 이상 현상

| 이상 현상 | 의미 |
|---|---|
| 삽입 이상 | 불필요한 데이터 없이는 삽입 불가 |
| 갱신 이상 | 일부만 수정돼 불일치 발생 |
| 삭제 이상 | 하나를 지우다 필요한 정보까지 삭제 |

## 2. 함수 종속

속성 집합 `X`를 알면 `Y`가 유일하게 결정될 때 `X -> Y` 라고 쓴다.

- 결정자: `X`
- 종속자: `Y`

예:

- `학번 -> 이름`
- 학번을 알면 이름이 정해진다.

## 3. 정규화 단계

| 단계 | 핵심 기준 |
|---|---|
| 1NF | 원자값 |
| 2NF | 부분 함수 종속 제거 |
| 3NF | 이행 함수 종속 제거 |
| BCNF | 모든 결정자가 후보키 |
| 4NF | 다치 종속 제거 |
| 5NF | 조인 종속 제거 |

## 4. 1NF ~ 3NF 이해

### 1NF

- 한 칸에 여러 값이 들어가면 안 된다.
- 모든 속성값이 원자값이어야 한다.

### 2NF

- 기본키가 복합키일 때 중요하다.
- 키의 일부에만 종속되는 일반 속성을 제거한다.

예:

- 기본키가 `(학번, 과목코드)`
- `이름` 이 `학번` 에만 종속
- 이 경우 부분 함수 종속이므로 2NF 위반

### 3NF

- 기본키가 아닌 속성 사이의 이행 종속 제거

예:

- `학번 -> 학과코드`
- `학과코드 -> 학과명`
- 따라서 `학번 -> 학과명` 이행 종속 발생

## 5. BCNF

3NF보다 더 강한 형태다.

- 모든 결정자가 후보키여야 한다.
- "결정자이지만 후보키가 아닌 속성"이 남아 있으면 BCNF 위반

## 6. 4NF와 5NF

- 4NF: 다치 종속 제거
- 5NF: 조인 종속 제거

실기에서는 1NF~BCNF 비중이 훨씬 크고, 4NF/5NF는 개념형으로 나오는 편이다.

## 7. 반정규화

정규화는 무결성에 유리하지만 조인이 많아질 수 있다. 성능 향상을 위해 의도적으로 중복을 허용하는 것을 반정규화라고 한다.

### 반정규화가 필요한 경우

- 조회 성능을 높여야 할 때
- 조인이 너무 많아 응답이 느릴 때
- 집계 결과를 자주 재사용할 때

## 8. 문제 추적 예제

### 예제 1: 2NF 판단

테이블: `(학번, 과목코드, 이름, 과목명, 점수)`

가정:

- `학번 -> 이름`
- `과목코드 -> 과목명`
- `(학번, 과목코드) -> 점수`

판단:

- 기본키는 `(학번, 과목코드)`
- 이름은 학번에만 종속 -> 부분 함수 종속
- 과목명은 과목코드에만 종속 -> 부분 함수 종속
- 따라서 2NF 위반

### 예제 2: 3NF 판단

`사원번호 -> 부서코드`
`부서코드 -> 부서명`

- 부서명은 사원번호에 직접 종속되지 않고 중간 속성(부서코드)을 통해 결정
- 이행 함수 종속
- 3NF 위반

## 9. 자주 틀리는 함정

### 9.1 2NF는 복합키일 때 특히 중요

- 기본키가 단일키라면 부분 함수 종속 문제가 발생하지 않는다.

### 9.2 3NF와 BCNF 혼동

- 3NF는 이행 종속 제거
- BCNF는 모든 결정자가 후보키여야 함

### 9.3 정규화와 반정규화를 선악처럼 이해

- 정규화는 무결성 중심
- 반정규화는 성능 중심
- 목적이 다르다.

## 10. 시험장에서 푸는 순서

1. 기본키가 단일키인지 복합키인지 먼저 본다.
2. 속성 간 종속 관계를 화살표로 적는다.
3. 부분 종속인지, 이행 종속인지 구분한다.
4. 해당 종속성을 제거하면 어느 정규형으로 가는지 판단한다.
5. 성능 목적의 중복 허용은 반정규화로 연결한다.

## 한 페이지 요약

- 정규화의 목적은 중복 제거와 이상 현상 방지다.
- 1NF는 원자값, 2NF는 부분 종속 제거, 3NF는 이행 종속 제거다.
- BCNF는 모든 결정자가 후보키여야 한다.
- 4NF/5NF는 다치 종속과 조인 종속 제거다.
- 반정규화는 성능 향상을 위해 의도적 중복을 허용하는 것이다.
