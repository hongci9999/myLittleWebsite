## 구성 요약

| 경로 | 설명 |
|------|------|
| `SQL_프로그래밍/` | 실기 **SQL** 목차별 노트 (DDL → DML → DCL·트랜잭션 → 함수·뷰 → 연습) |
| `Templates/` | Obsidian 등에서 복사해 쓰는 **SQL 노트 템플릿** |

### SQL 목차 (권장 순서)

1. `01_DDL` — 테이블 관리, 제약조건  
2. `02_DML` — SELECT 기본, 집계·GROUP BY, 조인·서브쿼리  
3. `03_DCL_및_트랜잭션` — GRANT/REVOKE, COMMIT/ROLLBACK  
4. `04_기타` — 내장 함수, 뷰, **DBMS·실기 문법 참고** (`02_DBMS_및_실기_문법_참고.md`)  
5. `99_Practice_Problems` — 기출 정리·오답 노트, **조인·서브쿼리 통합 예제** (`01_Join_서브쿼리_통합예제.md`)  

보강: `02_DML/03_Join_and_Subquery.md`에 IN/ANY/ALL/EXISTS **비교표·함정** 절 추가.
