# 포인터
## 포인터 연산
주소값
특정 데이터에 접근할 수 있는 기능
원본 데이터에 접근
& 주소값
* 해당 위치로 접근

포인터 변수
크기는 고정되어있음
포인터 변수의 자료형은 참조할 데이터의 자료형을 의미

int data = 10;
int * p; 포인터변수 p 선언
p = &data 주소값 할당
. * p = 20; p가 가르키는 변수에 20 할당

참조에 의한 전달 call by refernece
참조값을 전달해 데이터 원본 가공
