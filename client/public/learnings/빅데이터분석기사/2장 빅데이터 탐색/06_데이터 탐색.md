# 데이터 탐색 (Data Exploration)

## 📌 핵심 요약
데이터의 의미를 찾기 위해 **통계**와 **시각화**를 통해 파악하는 과정입니다. EDA(탐색적 자료 분석)는 가설 검증보다는 데이터 자체의 특성을 발견하는 데 초점을 둡니다.

---

## 📝 상세 정리

### 1. 데이터 탐색 개요

#### ● EDA (Exploratory Data Analysis, 탐색적 자료 분석)

데이터의 의미를 찾기 위해 **통계**와 **시각화**를 통해 파악하는 분석 접근법입니다.

#### ● EDA의 4가지 주제

| 주제 | 설명 |
| :--- | :--- |
| **1) 저항성의 강조** | 자료 변동에 민감하지 않음 (극단값·이상치에 덜 영향받는 지표 활용) |
| **2) 잔차 계산** | 값들이 주경향(평균 등)으로부터 얼마나 벗어나 있는지 확인하는 척도 |
| **3) 자료변수의 재표현** | 원래 변수를 적당한 척도로 변환 (로그, 제곱근 등) |
| **4) 그래프를 통한 현시성** | 시각화를 통하여 효율적으로 파악 |

> [!TIP] 암기법: **저잔재현** (저항성, 잔차, 재표현, 현시성)
