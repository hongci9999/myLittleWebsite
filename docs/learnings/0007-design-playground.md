# 디자인 플레이그라운드

날짜: 2026-02-26
태그: [프론트, ux, 도구]

## 요약

디자인 플레이그라운드는 폰트·색상·컴포넌트 스타일을 실시간으로 비교하고 결정하는 도구다. 선택 결과를 마크다운으로 복사해 AI에게 전달하면 index.css와 컴포넌트에 반영할 수 있다.

---

## 접근 경로

- URL: `/design-playground`
- About 페이지 → "디자인 플레이그라운드 열기" 버튼

## 구성

### 1. 폰트

| 구분 | 옵션 |
|------|------|
| 본문 | 시스템 기본, Pretendard, Noto Sans KR, 나눔고딕, Geist, Inter, Spoqa Han Sans |
| 코드 | 시스템 모노, Consolas, Fira Code, JetBrains Mono, Geist Mono, Victor Mono, Source Code Pro, Cascadia Code |

### 2. 색상 테마

- **복수 선택 가능** (클릭으로 선택/해제)
- **첫 번째 선택**이 페이지에 적용
- light, dark, warm, cool, forest, navy, blue-orange, dark-warm, dark-forest, dark-rose, dark-slate, dark-oled 등 20+ 테마

### 3. 모서리·그림자·트랜지션

- **모서리**: 각지게(0), 기본(0.5rem), 둥글게(0.75rem), 크게(1rem), 필(9999px)
- **그림자**: 없음, 미세, 기본, 강함, 매우 강함
- **트랜지션**: 없음, 빠름(150ms), 기본(300ms), 느림(500ms), 매우 느림(700ms)

### 5. 선택 결과 복사

- "선택 항목 복사" 버튼 → 현재 선택이 마크다운 형식으로 클립보드에 복사
- AI에게 붙여넣어 index.css·컴포넌트 반영 요청

## 참고

- 모서리(radius)는 `--radius` CSS 변수로 루트에 설정되며, 미리보기 컴포넌트에 `rounded-[var(--radius)]` 적용
- 결정 후 이 페이지는 삭제 가능 (임시 도구)
