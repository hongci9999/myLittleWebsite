# C언어 기초 문법 및 데이터 입출력

## 📌 핵심 요약
C언어의 기본적인 특성과 변수 선언 규칙, 데이터 타입([[자료형]]), 그리고 효율적인 입출력을 위한 서식 지정자와 연산자 활용법을 다룹니다. 특히 연산자의 우선순위와 전치/후치 증감 연산자의 동작 원리를 상세히 정리합니다.

---

## 📝 상세 정리

## 1. C언어의 특징 및 기본 구조
- **특징:** 이식성이 뛰어나고 효율적이며, 고급 언어와 저급 언어의 장점을 모두 가짐. 다양한 연산자 기능을 제공함.
- **기본 구조:**
    - `main` 함수를 가장 먼저 호출함.
    - 블록 구조: 중괄호 `{}`를 사용하여 코드 범위를 감쌈.
    - 대소문자를 엄격히 구분함.
    - 모든 문장은 세미콜론 `;`으로 마무리함.

### 1.1 [[상수]](Constant) 표현
- **정수형 상수:**
    - 8진수: 숫자 앞에 `0`을 붙임 (예: `012`).
    - 16진수: 숫자 앞에 `0x`를 붙임 (예: `0xA`).
- **실수형 상수:** 지수 표현 시 `e`를 사용함 (예: `2e3` = $2 \times 10^3$).
- **문자형 상수:** 작은따옴표(`' '`)를 사용하며, 각 문자는 특정 코드값(ASCII)으로 구성됨.
- **문자열 상수:** 큰따옴표(`" "`)를 사용함.

---

## 2. 변수 선언 및 자료형

### 2.1 [[변수]] 선언 규칙
- **기본 명명 규칙:**
    - 변수명의 첫 글자는 영문자 또는 언더바(`_`)로 시작해야 함.
    - 변수명은 고유해야 하며, **[[예약어]]**를 사용할 수 없음.
- **[[헝가리안 표기법]](Hungarian Notation):**
    - 변수명 앞에 데이터 타입을 암시하는 접두어를 붙임.
    - **예시:** `int nScore;`, `char cGrade;`, `float fAvg;`
    - 단점: 데이터 타입이 바뀌면 이름을 수정하기 번거로움.

### 2.2 자료형([[Data Type]])
- **변수:** 데이터를 저장하는 공간 및 그 값 자체. 최초 사용 시 반드시 **초기화**가 필요함.
- **자료형:** 저장 공간의 크기와 형태를 정의함.

---

## 3. 서식 문자열 및 입출력

### 3.1 [[이스케이프 시퀀스]](Escape Sequence)
| 문자 | 기능 |
| :--- | :--- |
| `\n` | 줄 바꿈 (New Line) |
| `\t` | 탭 (Tab) 만큼 간격 띄우기 |
| `\b` | 커서를 한 칸 왼쪽으로 이동 (Backspace) |
| `\r` | 현재 행의 처음으로 이동 (Carriage Return) |
| `\\` | 역슬래시(`\`) 자체를 출력 |

### 3.2 서식 지정자
- `%d`: 10진수 정수
- `%o`: 8진수 / `%x`: 16진수
- `%u`: 부호 없는(Unsigned) 10진수 정수
- `%f`: 실수 / `%e`: 지수 형태 실수
- `%c`: 문자 하나 / `%s`: 문자열

### 3.3 표준 입출력 함수
- **입력 함수:**
    - `getchar()`: 문자 하나 입력.
    - `gets(문자배열)`: 문자열 입력.
    - `scanf(서식문자열, &변수)`: 서식에 맞춰 입력받아 주소(`&`)에 저장.
- **출력 함수:**
    - `putchar(문자)`: 문자 하나 출력.
    - `puts(문자열)`: 문자열 출력 후 자동 줄 바꿈.
    - `printf(서식문자열, 값)`: 지정된 서식대로 출력.

## 4. 연산자(Operator)

### 4.1 연산자 우선순위
괄호 안이 최우선이며, 전체적인 순서는 다음과 같습니다:
1. **단항 연산자** (`!`, `++`, `--`, `~` 등)
2. **산술 연산자** (`*`, `/`, `%` → `+`, `-`)
3. **시프트 연산자** (`<<`, `>>`)
4. **관계 연산자** (`<`, `>`, `==`, `!=`)
5. **비트 연산자** (`&`, `^`, `|`)
6. **논리 연산자** (`&&`, `||`)
7. **복합 대입 연산자** (`+=`, `*=`, `/=` 등)

### 4.2 결과 데이터 타입 결정
- 같은 타입끼리 연산 시 결과도 해당 타입임 (예: $int / int = int$).
- 다른 타입끼리 연산 시 **더 크고 정밀한 타입**으로 자동 변환되어 계산됨.

### 4.3 주요 연산자 종류
- **비트 논리 연산자:**
    - `&`(AND), `|`(OR), `^`(XOR), `~`(NOT)
    - `>>`: 비트를 우측으로 이동. (예: `8 >> 2`는 $1000_{(2)}$를 우측으로 2칸 옮겨 $0010_{(2)}$, 즉 2가 됨)
    - `<<`: 비트를 좌측으로 이동.
- **[[증감 연산자]]:**
    - **전치(`++a`):** 값을 1 증가시킨 **후** 문장 계산.
    - **후치(`a++`):** 문장을 계산한 **후** 값을 1 증가시킴.

> [!INFO] 💡 AI 보충 설명: 증감 연산자 예시
> ```c
> int a = 5, b, c;
> b = ++a; // a가 6이 된 후 b에 대입 (b=6, a=6)
> c = a++; // a(6)를 c에 대입한 후 a가 증가 (c=6, a=7)
> ```

## 🎯 빈출 키워드
- **@([[상수]] / [[변수]]):** 변하지 않는 값과 변하는 값의 차이 및 선언 규칙.
- **@([[서식 지정자]]):** `%d`, `%f`, `%s` 등 출력 형식 제어.
- **@([[비트 연산자]]):** `&`, `|`, `^` 및 시프트(`<<`, `>>`) 연산의 원리.
- **@([[연산자 우선순위]]):** 복합 연산식에서 계산 순서 결정.
- **@([[증감 연산자]]):** 전치와 후치의 계산 결과 차이.
