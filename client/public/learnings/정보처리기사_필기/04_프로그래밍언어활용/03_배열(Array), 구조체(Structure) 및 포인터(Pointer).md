# 배열(Array), 구조체(Structure) 및 포인터(Pointer)의 핵심 원리

## 📌 핵심 요약
동일한 자료형의 집합인 **[[배열]]**과 서로 다른 자료형을 그룹화하는 **[[구조체]]**의 차이를 명확히 하고, 메모리 주소를 직접 제어하는 **[[포인터]]**의 메커니즘을 학습합니다. 특히 포인터 연산 시 자료형 크기에 따른 주소 증감 원리는 시험에 반드시 출제되는 포인트입니다.

---

## 📝 상세 정리

### 1. 배열 (Array)
- **정의**: 동일한 데이터 타입의 변수들을 하나의 이름으로 묶어 메모리에 연속적으로 배치한 자료구조입니다.
- **주소 상수 (Address Constant)**: 배열의 이름은 배열의 첫 번째 요소(`arr[0]`)의 시작 주소값을 가지며, 상무이므로 값을 변경할 수 없습니다.
- **인덱스 (Index)**: 0부터 시작하는 상대적 위치값으로 데이터에 접근합니다.
- **2차원 배열**: 행(Row)과 열(Column)로 구성된 평면 구조입니다.
    - **//예시 1**: `int a[3] = {1, 2, 3, 4, 5, 6};` 
        > [!CAUTION] 주의: 배열 크기가 3인데 요소가 6개이면 컴파일 에러 또는 오버플로우가 발생합니다.
    - **//예시 2**: `int a[2][3] = {{1, 2, 3}, {4, 5, 6}};` (2행 3열 구조)

### 2. 구조체 (Structure)
- **정의**: 사용자 정의 자료형으로, 성격이 다른 여러 데이터 타입을 하나의 단위로 묶어 관리합니다.
- **선언 및 이용 예시**:
```c
struct Person {
    char name[10];
    int age;
};

struct Person p1 = {"Kim", 25};
printf("%d", p1.age); // 점(.) 연산자로 멤버 접근
```

### 3. 포인터 (Pointer)
- **개념**: 메모리 주소값을 저장하는 변수입니다.
- **연산자**:
    - `&` (Address-of): 변수의 주소를 추출합니다.
    - `*` (Dereference): 포인터가 가리키는 주소의 실제 데이터에 접근합니다.
- **사용 예시**:
```c
int data = 10;
int *p = &data; // p에 data 주소 할당
*p = 20;        // 역참조를 통해 data의 값을 20으로 변경
```

### 4. 참조에 의한 전달 (Call by Reference)
- 함수 호출 시 인자로 값이 아닌 **메모리 주소**를 전달하여, 함수 내부에서 원본 데이터를 직접 가공하는 방식입니다.
- **//예시**:
```c
void multiply(int *n) {
    *n = (*n) * 2;
}
```

### 5. 포인터 연산 (Pointer Arithmetic)
- 주소값에 대한 가감 연산 시, 해당 포인터 자료형의 크기($sizeof$)만큼 곱하여 계산됩니다.
- $New\_Address = Current\_Address + (n \times sizeof(Type))$
- **//예시**: `int *p = 100;`일 때 `p + 1`의 주소는 `104`가 됩니다. (int가 4바이트인 경우)

### 6. 2차원 배열과 포인터의 관계
- `arr[i][j]`는 포인터 표현식 `*(*(arr + i) + j)`와 완벽히 동일합니다.
- **//예시**: `a[2][3]`에서 `a[1]`은 두 번째 행의 시작 주소를 의미하는 포인터 역할을 합니다.

---

> [!INFO] 💡 AI 보충 설명
> 1. **배열과 포인터의 표기법 호환성**: `p[i]`는 `*(p + i)`와 같습니다. 이는 포인터 변수에도 배열 인덱스 기호를 쓸 수 있음을 의미합니다.
> 2. **NULL 포인터**: 아무것도 가리키지 않는 포인터는 `NULL`로 초기화하여 메모리 오염을 방지하는 것이 전공 지식상 권장됩니다.
> 3. **구조체와 화살표 연산자**: 구조체 포인터 `ps`가 있을 때, `(*ps).age`는 `ps->age`와 같습니다.

## 🎯 빈출 키워드
- **[[배열]] (Array)**: 동일 타입의 연속적 나열. 이름은 첫 요소의 주소(상수).
- **[[구조체]] (Structure)**: `struct` 키워드를 사용한 사용자 정의 복합 데이터 타입.
- **[[포인터]] (Pointer)**: 메모리 주소를 담는 변수. `*` 연산자로 실제 값 참조.
- **[[Call by Reference]]**: 주소 전달 방식. 원본 데이터 수정 가능.
- **[[포인터 연산]]**: `p + n` 수행 시 자료형 크기만큼 주소가 점프하는 원리.
