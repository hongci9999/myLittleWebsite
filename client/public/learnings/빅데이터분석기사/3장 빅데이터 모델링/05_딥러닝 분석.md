# 딥러닝 분석 (Deep Learning Analysis)

> [!NOTE] 관련 문서
> 인공신경망 기초(퍼셉트론, MLP, 역전파 등)는 [03_분석기법 적용_2.md](./03_분석기법%20적용_2.md) 참조.

## 📌 핵심 요약
**DNN**은 **2개 이상의 은닉층**을 가진 인공신경망입니다. **CNN**은 Convolution·Pooling으로 이미지 패턴을 추출합니다. **RNN**은 **순차 데이터** 학습에 특화되며, **LSTM·GRU**로 장기 의존성 문제를 해결합니다. **오토인코더**는 Encoder→**Context Vector**→Decoder 구조로, **VAE·GAN** 등 생성형 AI의 기반 모델입니다. **Seq2Seq**는 입력·출력 길이가 달라도 변환 가능합니다. **트랜스포머**는 느린 속도·병렬처리 불가·정보 손실을 개선한 **Attention** 기반 모델이며, **BERT**(인코더·양방향)·**GPT**(디코더·일방향)가 대표적입니다.

---

## 📝 상세 정리

### 1. DNN (심층 신경망, Deep Neural Network)

**정의:** **2개 이상의 은닉층**을 가진 인공신경망.

**구조:** 입력층 → 은닉층(복수) → 출력층

---

### 2. CNN (합성곱 신경망, Convolutional Neural Network)

**정의:** **Convolution**과 **Pooling** 층을 사용해 **이미지 패턴**을 찾는 신경망.

**구조:** 입력 → **Convolution** → **Pooling** → **Flatten** → **Fully Connected** → 출력

| 층 | 설명 |
| :--- | :--- |
| **Convolution Layer** | **합성곱 연산**(=Filtering)으로 이미지 **특징 추출** |
| **Pooling Layer** | 데이터 크기 **축소**, 계산량 감소, **공간적 특성** 유지 |
| **Flatten** | 2D·3D 데이터를 **1D**로 변환 |
| **Fully Connected Layer** | **DNN** 구조로 분류·회귀 등 **최종 작업** 수행 |

---

### 3. 주요 CNN 모델

| 용도 | 모델 |
| :--- | :--- |
| **분류 (Classification)** | LeNet, AlexNet, VGG Nets, GoogLeNet, ResNet, EfficientNet |
| **객체탐지 (Object Detection)** | RCNN, SPP Net, **YOLO**, Attention Net, EfficientDet |
| **분할 (Segmentation)** | FCN, DeepLab, U-net, SegNet |

---

### 4. RNN (순환 신경망, Recurrent Neural Network)

**정의:** **순환 구조**를 가진 신경망. **순차 데이터** 학습에 특화.

#### ● 장기 의존성 문제 (Long-term Dependency)

과거 은닉층의 정보가 효과적으로 전달되지 않는 현상.

#### ● 장기 의존성 극복 모델

| 모델 | 특징 |
| :--- | :--- |
| **LSTM** | **Forget Gate**, **Input Gate**, **Output Gate** |
| **GRU** | **Reset Gate**, **Update Gate** |

---

### 5. 오토인코더 (Autoencoder)

**정의:** **비지도 학습** 신경망. **Encoder**로 입력을 압축하고, **Decoder**로 복원합니다.

**구조:** Encoder → **Context Vector (=Latent Space)** → Decoder

#### ● 활용

**1) 생성 모델 (Generative Models)**

| 모델 | 설명 |
| :--- | :--- |
| **VAE** | **확률분포**를 학습하여 데이터 생성 |
| **GAN** | **Generator**와 **Discriminator**의 **대립 학습(Adversarial)**로 유사 데이터 생성 |
| **DCGAN** | GAN + CNN 결합. **안정적 학습** |

> [!NOTE] 생성형 AI
> 오토인코더는 **생성형 AI**의 기반 모델.

**2) 이상 탐지 (Anomaly Detection)**  
정상 데이터만 학습하여, 이상 데이터를 식별·판단.

---

### 6. Seq2Seq (Sequence-to-Sequence)

**정의:** **Encoder**와 **Decoder**를 사용해 한 문장을 다른 문장으로 변환하는 모델.

| 특징 | 설명 |
| :--- | :--- |
| **입·출력 길이** | **입력과 출력의 길이가 달라도** 변환 가능 |
| **한계** | 길이가 긴 문장에서 **정보 손실** 발생 |

---

### 7. 트랜스포머 (Transformer)

**정의:** **느린 속도**, **병렬처리 불가**, **정보 손실** 등의 단점을 개선한 **Attention** 기반 모델.

#### ● Attention

시퀀스의 **각 요소에 가중치**를 부여하여 **중요 정보에 집중**하고, **장기 의존성 문제**를 해결.

#### ● 구성요소

**Positional Encoding**, **Self-Attention**, **Feed Forward Network**

#### ● 주요 모델

| 모델 | 개발 | 구조 | 학습 방식 | 방향 |
| :--- | :--- | :--- | :--- | :--- |
| **BERT** | 구글 | **인코더** | 문장 중간 빈칸 채우기 (MLM) | **양방향** |
| **GPT** | OpenAI | **디코더** | 이전 단어로 다음 단어 예측 | **일방향** |
