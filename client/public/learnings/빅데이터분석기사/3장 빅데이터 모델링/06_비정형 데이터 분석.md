# 비정형 데이터 분석 (Unstructured Data Analysis)

## 📌 핵심 요약
**단어 표현**은 희소표현(원핫)과 카운트 기반(BOW, TF-IDF, LSA)으로 나뉩니다. **워드 임베딩**(Word2Vec, GloVe, FastText, ELMo)은 밀집 벡터로 의미를 반영합니다. **비정형 분석 기법**으로 소셜 네트워크 분석(중심성), 감정분석, 오피니언 마이닝, 리얼리티 마이닝이 있습니다.

---

## 📝 상세 정리

### 1. 단어 표현 (Word Representation)

#### ● 희소표현 (Sparse Representation)

| 방법 | 설명 |
| :--- | :--- |
| **원핫 인코딩 (One-hot)** | 단어별 고유 벡터 (예: [1,0,0], [0,1,0]). **메모리 낭비**, **의미 관계 미포착** |

#### ● 카운트 기반 (Count-based)

| 방법 | 설명 |
| :--- | :--- |
| **BOW (Bag of Words)** | 단어 **출현 횟수** 기반 표현 |
| **TDM (Term-Document Matrix)** | 문서별 단어 빈도를 **행렬**로 표현 |
| **TF-IDF** | **TF**(단어 빈도) × **IDF**(역문서 빈도) |
| **LSA (Latent Semantic Analysis)** | **SVD**(특이값 분해)로 잠재 의미·차원 축소 |

---

### 2. 워드 임베딩 (Word Embedding)

밀집 벡터로 단어를 표현. **의미·유사도** 반영.

| 모델 | 설명 |
| :--- | :--- |
| **Word2Vec** | 거리 기반 벡터. **CBOW**: 주변 단어 → 중심 단어 추론. **Skip-Gram**: 중심 단어 → 주변 단어 예측 |
| **GloVe** | 전체 문장·말뭉치의 **통계 정보** 반영 |
| **FastText** | 단어를 **서브워드** 단위로 분해하여 벡터 계산 |
| **ELMo** | **양방향** 언어 모델 적용 |

---

### 3. 비정형 데이터 분석 기법

#### ● 소셜 네트워크 분석 (Social Network Analysis)

**노드**와 **엣지**로 표현. SNS에서 **영향력 있는 인물** 탐색 등에 활용.

**중심성 (Centrality):**

| 유형 | 설명 |
| :--- | :--- |
| **정점도 중심성 (Degree)** | 연결된 노드 수 |
| **매개도 중심성 (Betweenness)** | 다른 노드 쌍 간 경로에 얼마나 개입하는지 |
| **고유벡터·근접도 중심성** | Eigenvector, Closeness |

#### ● 기타 기법

| 기법 | 설명 |
| :--- | :--- |
| **감정분석 (Sentiment Analysis)** | 텍스트의 **긍정·부정** 감정 분석 |
| **오피니언 마이닝 (Opinion Mining)** | 특정 이슈에 대한 **의견** 분석 |
| **리얼리티 마이닝 (Reality Mining)** | 기계·센서 등에서 **비정형 데이터** 추출·분석 |
