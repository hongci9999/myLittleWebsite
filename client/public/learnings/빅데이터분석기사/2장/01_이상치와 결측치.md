# 데이터 정제 (Data Cleaning)

## 📌 핵심 요약
데이터 분석 전 반드시 수행해야 하는 단계로, 기록되지 않은 값인 **결측치(Missing Value)**와 데이터의 일반적인 패턴에서 현저히 벗어난 **이상치(Outlier)**를 탐지하고 처리합니다. 이는 분석 결과의 왜곡을 방지하고 모델의 성능을 향상시키는 데 목적이 있습니다.

---

## 📝 상세 정리

### 1. 결측치 (Missing Value)
데이터셋에서 특정 변수의 값이 기록되지 않은 상태를 의미합니다.

* **발생 원인:**
    * **데이터 수집 오류:** 센서 고장, 전송 누락 등 기계적 결함.
    * **의도적인 미응답:** 설문 조사 시 민감한 질문에 대해 답변을 거부한 경우.
    * **측정 불가능:** 특정 조건에서 값을 측정할 수 없는 경우.
* **처리 방법:**
    * **삭제 (Deletion):** 결측치가 포함된 행(레코드)이나 열(변수)을 제거.
    * **대체 (Imputation):** 평균(Mean), 중앙값(Median), 최빈값(Mode) 등으로 채움.
    * **예측 (Prediction):** 다른 변수들과의 관계를 이용하여 결측값을 추정.
    * **모델링 기법:** $K$-최근접 이웃($K$-Nearest Neighbor, KNN), 다중 대체법(Multiple Imputation) 등 활용.

### 2. 이상치 (Outlier)
관측된 데이터 중 다른 데이터들과 현저하게 차이가 나는 값을 의미합니다.

* **발생 원인:**
    * **입력 오류:** 수치 입력 시 오타(예: 10을 1000으로 입력).
    * **자연스러운 변동:** 실제 현상에서 아주 드물게 발생하는 극단적인 값.
    * **데이터 처리 오류:** 데이터 변환이나 결합 과정에서 발생하는 왜곡.
* **탐지 방법:**
    * **통계적 방법:** * 평균 기준: $평균 \pm 3 \times 표준편차(\sigma)$ 범위를 벗어나는 값 ($Z$-Score).
        * 사분위수 기준: $IQR(Interquartile \ Range)$ 방식을 활용하여 탐지.
    * **시각적 방법:** 박스플롯(Box Plot), 산점도(Scatter Plot) 등을 통해 시각적으로 확인.



* **처리 방법:**
    * **삭제:** 분석 목적에 맞지 않는 오류성 이상치는 제거.
    * **값 대체:** 상한값이나 하한값으로 보정하거나 평균 등으로 대체.
    * **변환 (Transformation):** 로그 변환($\log$), 제곱근 변환($\sqrt{x}$) 등을 통해 데이터의 치우침(Skewness)을 완화.

> [!INFO] 💡 AI 보충 설명
> 
> **1. 범주형(형태형) 데이터의 결측치 처리**
> 수치형과 달리 평균을 낼 수 없으므로 주로 다음 방법을 사용합니다.
> * **최빈값 대체:** 가장 자주 등장하는 항목으로 채움.
> * **새로운 범주 생성:** '알 수 없음(Unknown)'이라는 별도의 카테고리로 분류.
> * **예측 모델:** 분류 알고리즘(예: Logistic Regression, Random Forest)을 사용하여 적절한 카테고리를 예측.
> 
> **2. 자가회귀(Autoregression)를 이용한 결측치 대체**
> 시계열 데이터에서 유용한 방법입니다.
> * 이전 시점의 데이터들이 현재 시점의 데이터에 영향을 준다는 가정하에, 과거의 관측값을 독립변수로 하여 결측 시점의 값을 예측합니다.
> * 데이터의 추세(Trend)나 계절성(Seasonality)이 뚜렷할 때 효과적입니다.
> 
> **3. 일변량 분석(Univariate Analysis)이란?**
> * **정의:** 단 하나의 변수(Feature)에 대해서만 데이터의 특성을 파악하는 분석 기법입니다.
> * **목적:** 변수의 분포, 중심 경향성(평균, 중앙값), 분산도(표준편차, 범위) 등을 파악하여 데이터의 기본적인 성질을 이해하는 것입니다.
> * **방법:** 히스토그램, 박스플롯, 빈도 분석 등이 주로 쓰입니다.
