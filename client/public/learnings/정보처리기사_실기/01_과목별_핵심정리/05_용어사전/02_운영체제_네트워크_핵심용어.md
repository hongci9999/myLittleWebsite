# 운영체제/네트워크 핵심용어

- 프로세스(Process) = 실행 중인 프로그램
- 스레드(Thread) = 프로세스 내부 실행 흐름 단위
- 반환시간(Turnaround Time) = 제출부터 완료까지 걸린 전체 시간
- 대기시간(Waiting Time) = 준비 큐에서 기다린 시간
- 커널(Kernel) = 운영체제의 핵심 자원 관리 부분
- 쉘(Shell) = 사용자 명령 해석기
- 페이징(Paging) = 페이지와 프레임 단위의 메모리 관리
- 스래싱(Thrashing) = 페이지 교체가 과도해 성능이 급락하는 현상
- 프로토콜(Protocol) = 통신 규약
- OSI 7계층 = 통신 기능을 7개 계층으로 나눈 참조 모델
- TCP = 연결형, 신뢰성 있는 전송 프로토콜
- UDP = 비연결형, 빠른 전송 프로토콜
- IP = 패킷 전달 프로토콜
- ARP = IP 주소를 MAC 주소로 변환
- ICMP = 오류 보고 및 상태 진단 프로토콜
- 서브네팅(Subnetting) = 네트워크를 더 작은 서브넷으로 분할
- NAT = 사설 IP를 공인 IP로 변환하는 기술

## 한 페이지 요약

- 운영체제는 자원 관리, 네트워크는 계층/프로토콜/주소 체계 중심으로 정리한다.
