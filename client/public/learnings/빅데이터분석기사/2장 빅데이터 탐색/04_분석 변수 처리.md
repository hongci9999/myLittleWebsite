# 분석 변수 처리

> 이 문서는 **변수 선택**, 변수 변환, 변수 생성, 상관계수 분석 등 분석에 사용할 변수를 다루는 처리 전반을 포함합니다.

## 📌 핵심 요약

분석 성능 향상을 위해 **중요한 변수만 선별**하거나, 변수를 변환·생성하는 과정입니다. 통계적 측정치, 모델 성능, 또는 학습 과정 내 규제를 활용하여 변수를 선택합니다.

---

## 📝 상세 정리

### 변수 선택 방법

- **(1) 필터(Filter) 방법**
  - **설명:** 통계적 측정치를 활용하여 변수 선택.
  - **기법:** 상관계수, 카이제곱 검정, F-검정, 상호정보량(Mutual Information).

- **(2) 래퍼(Wrapper) 방법**
  - **설명:** 모델의 성능을 기준으로 변수 선택.
  - **기법:** 전진 선택법, 후진 제거법, 단계적 선택법.

- **(3) 임베디드(Embedded) 방법**
  - **설명:** 모델 학습 과정에서 변수 선택.
  - **기법:** Lasso 규제, Ridge 규제.

---

### 상관관계 분석 (Correlation Analysis)

두 변수 간의 **선형적 관계** 존재 여부를 파악하는 분석입니다.

#### ● 상관분석 유형

| 유형 | 설명 |
| :--- | :--- |
| **단순상관분석** | 두 개의 변수가 어느 정도 강한 관계에 있는지 측정 |
| **다중상관분석** | 3개 이상의 변수 간 관계를 측정 |
| **편상관관계분석** | 제3의 변수를 통제한 상태에서 두 변수의 상관관계 분석 |

#### ● 상관분석 방법

| 방법 | 적용 대상 | 특징 |
| :--- | :--- | :--- |
| **피어슨 (Pearson)** | 양적 척도, 연속형 변수 | 선형관계 크기 측정 |
| **스피어만 (Spearman)** | 서열 척도, 순서형 변수 | 선형/비선형 관계 나타냄 (단조 관계) |

> [!CAUTION] ⚠️ 주의사항
> - **스피어만:** X와 Y가 선형 관계가 아니더라도 +1 또는 -1이 될 수 있음 (단조 관계).
> - **통계적 유의성:** 상관계수 값으로는 **정도(강도)**만 파악 가능. 통계적으로 유의한지는 통계적 검정을 수행해야 함.

#### ● 상관계수 매트릭스 분석

|        | X1    | X2   | X3    | X4    |
| ------ | ----- | ---- | ----- | ----- |
| **X1** | 1.0   | 0.21 | -0.31 | 0.95  |
| **X2** | 0.21  | 1.0  | 0.02  | 0.13  |
| **X3** | -0.31 | 0.02 | 1.0   | -0.05 |
| **X4** | 0.95  | 0.13 | -0.05 | 1.0   |

- **해석:** X1과 X4가 높은 상관계수(0.95)를 가지므로, 둘 중 하나의 변수를 제거하는 것이 좋습니다.
- **상관계수 범위:** -1 ~ 1. -1은 음의 상관관계, 1은 양의 상관관계를 의미합니다.

### 파생변수 생성

- **요약변수 vs 파생변수**
  - **요약변수 (Summary Variable):** 수집된 정보를 통합한 변수로, 재사용성이 높음. (예: 1개월간 수입)
  - **파생변수 (Derived Variable):** 의미를 부여한 변수로, 논리적 타당성이 필요함. (예: 고객구매등급)

- **파생변수 생성 방법**
  - 특징 추출 (Feature extraction)
  - 결합 (Combination)
  - 부가적 정보 결합 (Combination of additional information)
  - 수학적 변환 (Mathematical transformation)
  - 교호작용의 반영 (Reflection of interaction effects)

- **교호작용 (Interaction Effect):** 두 개 이상의 변수가 상호작용을 하여, 특정 변수에 영향을 미치는 경우.
